# startupjs routes-middleware
> Server-side middleware for rendering client apps

## Installation

```sh
yarn add @startupjs/routes-middleware
```

## Requirements

```
react: >= 16.9.0 < 17
```

## Usage

```js
import '@startupjs/routes-middleware'
```
