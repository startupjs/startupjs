# Кеширование node_modules

Чтобы включить кэширование `node_modules` в проекте, напишите следующие команды в папке проекта:

1. Отключите глобальный кэш для `yarn` и он сохранит кэш пакетов в локальной папке вашего проекта (по умолчанию `.yarn / cache`), которую затем можно добавить в Git

    ```
    yarn config set enableGlobalCache false
    ```

2. Отключите игнорирование папки кэша из git

    ```
    echo '!.yarn/cache' >> .gitignore
    ```
