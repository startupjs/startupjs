# Sidebar

Sidebar provide access to navigation links. They can either be permanently on-screen or controlled.


## Example

```jsx
import { Sidebar } from '@startupjs/ui'
```

```jsx example
const PATH = '_session.ExampleSidebar'
const $opened = useModel(PATH)
useEffect(() => {
  $opened.set(true)
}, [])
return (
    <Sidebar
      path={PATH}
      renderContent={() => (
        <Div>
          <Br />
          <Menu>
            <Menu.Item onPress={() => null}>Nav-1</Menu.Item>
            <Menu.Item onPress={() => null}>Nav-2</Menu.Item>
            <Menu.Item onPress={() => null}>Nav-3</Menu.Item>
            <Menu.Item onPress={() => null}>Nav-4</Menu.Item>
            <Menu.Item onPress={() => null}>Nav-5</Menu.Item>
            <Menu.Item onPress={() => null}>Nav-6</Menu.Item>
          </Menu>
        </Div>
      )}
    >
      <Content
        padding
        width='desktop'
        style={{
          backgroundColor: 'white'
        }}
      >
        <Row>
          <Button variant='flat' onPress={() => $opened.set(!$opened.get())}>
            Toggle sidebar
          </Button>
        </Row>
```

## Position

The `position` property specifies the side of the window from which the sidebar will slide in. Possible values of `position` property are `left` (default) and `right`.

```jsx
import { Sidebar } from '@startupjs/ui'
```

```jsx example
const PATH = '_session.PositionExampleSidebar'
const $opened = useModel(PATH)
useEffect(() => {
  $opened.set(true)
}, [])
return (
    <Sidebar
      path={PATH}
      position='right'
      renderContent={() => (
        <Div>
          <Br />
          <Menu>
            <Menu.Item onPress={() => null}>Nav-1</Menu.Item>
            <Menu.Item onPress={() => null}>Nav-2</Menu.Item>
            <Menu.Item onPress={() => null}>Nav-3</Menu.Item>
```

## Locked Sidebar

Specifying `forceClosed` property will keep the drawer closed.


```jsx
import { Sidebar } from '@startupjs/ui'
```

```jsx example

useEffect(() => {
  $opened.set(true)
}, [])
```
