# LineSeparator

LineSeparator is used to separate content or groups of content in a list or layouts.

## Example

```jsx
import { LineSeparator } from '@startupjs/ui'
```

```jsx example
return (
  <Content padding style={{ backgroundColor: 'white' }}>
    <Span description size='l'>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
    </Span>
    <Br />
    <LineSeparator>
      <Span bold>Example separator</Span>
    </LineSeparator>
    <Br />
    <Span description size='l'>
      Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.
    </Span>
  </Content>
)
```

## Left & right alignment.

Position of the text can be modified using `align` prop. Default value of `align` is `center`.

```jsx
import { LineSeparator } from '@startupjs/ui'
```

```jsx example
return (
  <Content padding style={{ backgroundColor: 'white' }}>
    <Span description size='l'>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
    </Span>
    <Br />
    <LineSeparator align='left'>
      <Span bold>Text Left</Span>
    </LineSeparator>
    <Br />
    <Span description size='l'>
      Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.
    </Span>
    <Br />
    <LineSeparator align='right'>
      <Span bold>Text Right</Span>
    </LineSeparator>
    <Br />
    <Span description size='l'>
      Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.
    </Span>
  </Content>
)
```
