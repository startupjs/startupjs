# Toast

Компонент для показа уведомлений
Для использования нужно использовать функцию toast

```js
import { toast } from '@startupjs/ui'
```

## Простой пример

- title (string): заголовок
- text (string): описание
- alert (boolean): убирает закрытие через 5 секунд если true
- icon (ReactNode): иконка
- type ('info' | 'error' | 'warning' | 'success'): задает опреденные стили для каждого типа
- actionLabel (string): текст для кнопки
- onAction (function): калбэк функция, вызывается при нажатии на кнопку action
- onClose (function): калбэк функция, вызывается при закрытии

```jsx example
function onShow () {
  toast({
    title: 'Заголовок',
    text: 'Описание',
    onAction: ()=> alert('onAction'),
    onClose: ()=> alert('onClose')
  })
}

return pug`
  Button(onPress=onShow) Показать уведомление
`
```
