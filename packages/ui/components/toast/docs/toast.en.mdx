# Toast

Component for showing notifications
To use, you need to use the toast function

```js
import { toast } from '@startupjs/ui'
```

## Simple example

- title (string): title
- text (string): description
- alert (boolean): removes closing after 5 seconds if true
- icon (ReactNode): icon
- type ('info' | 'error' | 'warning' | 'success'): sets specific styles for each type
- actionLabel (string): text for the button
- onAction (function): callback function, called when the action button is pressed
- onClose (function): callback function, called on close

```jsx example
function onShow () {
  toast({
    title: 'Title',
    text: 'Text',
    onAction: ()=> alert('onAction'),
    onClose: ()=> alert('onClose')
  })
}

return pug`
  Button(onPress=onShow) Show notification
`
```
