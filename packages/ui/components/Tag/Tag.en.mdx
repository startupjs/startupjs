# Tag

Tags allows developers to display any markup in separate blocks.

## Example


```jsx
import { Tag, Row } from '@startupjs/ui'
```
```jsx example
return (
  <Row>
    <Tag
      color='blue'
      textColor='white'
      style={{ marginRight: 10 }}
    >
      TAG-1
    </Tag>
    <Tag
      color='blue'
      textColor='white'
    >
      TAG-2
    </Tag>
  </Row>
)
```

## Variant

`variant` prop can be either `flat` or `outlined`. Default variant is `flat`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```
```jsx example
return (
  <Row>
    <Tag
      color='blue'
      style={{ marginRight: 10 }}
    >
      FLAT
    </Tag>
    <Tag
      color='blue'
      variant='outlined'
    >
      OUTLINED
    </Tag>
  </Row>
)
```


## Shape

`shape` prop can be either `circle` or `rounded`. Default shape is `circle`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```
```jsx example
return (
  <Row>
    <Tag
      color='blue'
      style={{ marginRight: 10 }}
    >
      CIRCLE
    </Tag>
    <Tag
      color='blue'
      shape='rounded'
    >
      ROUNDED
    </Tag>
  </Row>
)
```

## Icon

Tag can also have a icon using the `icon` props, `iconPosition` can be either `left` (default) or `right`.

Tag can be made interactive using `onIconPress` prop.


```jsx
import { Tag, Row } from '@startupjs/ui'
```
```jsx example
const [showTags, setShowTags] = useState([true, true])
const style = {
  marginRight: 10,
  paddingTop: 15,
  paddingBottom: 15 
}
return (
  <Row>
    <Tag
      color='blue'
      style={style}
      icon={() => (
        <FontAwesomeIcon icon={faTimesCircle} color='white' />
      )}
    >
      <Span style={{ color: 'inherit' }} size='l'> Not Interactive </Span>
    </Tag>
    {showTags[0] &&
      <Tag
        color='blue'
        style={{ marginRight: 10, paddingTop: 15, paddingBottom: 15 }}
        icon={() => (
          <FontAwesomeIcon icon={faTimesCircle} color='white' />
        )}
        onIconPress={() => setShowTags(arrTags => [!arrTags[0], arrTags[1]])}
      >
        <Span style={{ color: 'inherit' }} size='l'> Interactive Left </Span>
      </Tag>
    }
    {showTags[1] &&
      <Tag
        color='blue'
        style={{ marginRight: 10, paddingTop: 15, paddingBottom: 15 }}
        icon={() => (
          <FontAwesomeIcon icon={faTimesCircle} color='white' />
        )}
        iconPosition='right'
        onIconPress={() => setShowTags(arrTags => [arrTags[0], !arrTags[1]])}
      >
        <Span style={{ color: 'inherit' }} size='l'> Interactive Right </Span>
      </Tag>
    }
  </Row>
)
```
