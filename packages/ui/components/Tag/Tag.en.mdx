# Tag

Tags allows developers to display any markup in separate blocks.

## Example


```jsx
import { Tag, Row } from '@startupjs/ui'
```
```jsx example
return (
  <Row>
    <Tag
      color='blue'
      textColor='white'
      style={{ marginRight: 10 }}
    >
      TAG-1
    </Tag>
    <Tag
      color='blue'
      textColor='white'
    >
      TAG-2
    </Tag>
  </Row>
)
```

## Variant

`variant` prop can be either `flat` or `outlined`. Default variant is `flat`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```
```jsx example
return (
  <Row>
    <Tag
      color='blue'
      style={{ marginRight: 10 }}
    >
      FLAT
    </Tag>
    <Tag
      color='blue'
      variant='outlined'
    >
      OUTLINED
    </Tag>
  </Row>
)
```


## Shape

`shape` prop can be either `circle` or `rounded`. Default shape is `circle`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```
```jsx example
return (
  <Row>
    <Tag
      color='blue'
      style={{ marginRight: 10 }}
    >
      CIRCLE
    </Tag>
    <Tag
      color='blue'
      shape='rounded'
    >
      ROUNDED
    </Tag>
  </Row>
)
```
