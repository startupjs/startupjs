# Tag
Inherits [Button props](/docs/en/components/Button).

Tags are compact elements that represent an input, attribute, or action. They allow users to enter information, make selections, filter content, or trigger actions. While buttons are expected to appear consistently and with familiar calls to action, tags should appear dynamically as a group of multiple interactive elements.

```jsx
import { Tag } from '@startupjs/ui'
```

## Example

```jsx example
return (
  <Row>
    <Tag>Some tag</Tag>
    <Tag pushed>Other tag</Tag>
  </Row>
)
```

## Colors

The `color` property is `primary` by default. It can be changed by changing `color` prop. Possible values of property can be found in the `Sandbox` section at the bottom of the page.

```jsx example
return (
  <Row>
    <Tag>Default primary</Tag>
    <Tag color='success' pushed>Success</Tag>
    <Tag color='warning' pushed>Warning</Tag>
    <Tag color='error' pushed>Error</Tag>
  </Row>
)
```

## Outlined tags

Outlined tags offer an alternative style by passing `outlined` string on `variant` property to component.

```jsx example
return (
  <Row>
    <Tag variant='outlined'>Outlined primary</Tag>
    <Tag color='error' variant='outlined' pushed>Outlined error</Tag>
  </Row>
)
```

## Shapes

Tags can have different corner shapes. It can be changed by changing `shape` prop (`circle` by default) to `rounded`.

```jsx example
return (
  <Row>
    <Tag>Circle shape</Tag>
    <Tag shape='rounded' pushed>Rounded shape</Tag>
  </Row>
)
```

## Icons

Icon can be added using `icon` prop. Position of icon can be changed by passing `iconPosition` to component (`left` by default).

```jsx example
return (
  <Row>
    <Tag icon={faCoffee}>Icon on the left</Tag>
    <Tag icon={faCoffee} iconPosition='right' pushed>Icon on the right</Tag>
  </Row>
)
```

## Actions
Tag can be made interactive using `onPress` prop.

```jsx example
const [counter, setCounter] = useState(0)
return (
  <Tag onPress={() => setCounter(counter + 1)}>
    Clicked {counter} times
  </Tag>
)
```
