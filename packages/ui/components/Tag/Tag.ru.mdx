# Tag (тег)
Наследует [Button Props](/docs/ru/Button).

Теги - это компактные элементы, представляющие входные данные, атрибут или действие. Они позволяют пользователям вводить информацию, делать выбор, фильтровать контент или инициировать действия. В то время как предполагается, что кнопки будут появляться с призывами к действию, теги должны появляться динамически как группа из нескольких интерактивных элементов.

## Пример

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Tag>Primary</Tag>
    <Tag color='success' pushed>Success</Tag>
    <Tag color='warning' pushed>Warning</Tag>
    <Tag color='error' pushed>Error</Tag>
  </Row>
)
```

## Контурные теги

Контурные теги предлагают альтернативный стиль, передвая `outlined` строку в свойство `variant` компонента.

```jsx example
return (
  <Row>
    <Tag variant='outlined'>Outlined primary</Tag>
    <Tag color='error' variant='outlined' pushed>Outlined error</Tag>
  </Row>
)
```

## Shapes

Теги могут иметь различную угловую форму. Её можно изменить, изменив свойство `shape` (`circle` по умолчанию) на `rounded`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Tag>Circle shape</Tag>
    <Tag shape='rounded' pushed>Rounded shape</Tag>
  </Row>
)
```

## Иконки

Иконка может быть добавлена используя `icon` свойство. Положение иконки можно изменить, передав в компонент `iconPosition` (`left` по умолчанию).

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Tag icon={faCoffee}>Icon on the left</Tag>
    <Tag icon={faCoffee} iconPosition='right' pushed>Icon on the right</Tag>
  </Row>
)
```

## Действия
Тег можно сделать интерактивным используя `onPress` свойство.

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
const [counter, setCounter] = useState(0)
return (
  <Tag onPress={() => setCounter(counter + 1)}>
    Нажато {counter} раз
  </Tag>
)
```
