# Tag
Наследует [Button Props](/docs/ru/Button).

Tags - это компактные элементы, представляющие входные данные, атрибут или действие. Они позволяют пользователям вводить информацию, делать выбор, фильтровать контент или инициировать действия. В то время как предполагается, что кнопки будут появляться с призывами к действию, теги должны появляться динамически как группа из нескольких интерактивных элементов.

## Примеры

### Variant

`variant` свойство может быть `flat` (по умолчанию) либо `outlined`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Tag>TAG-1</Tag>
    <Tag variant='outlined' pushed>TAG-1</Tag>
    <Tag color='error' pushed>TAG-2</Tag>
    <Tag color='error' variant='outlined' pushed>TAG-2</Tag>
    <Tag color='#6F00AC' pushed>TAG-3</Tag>
    <Tag color='#6F00AC' variant='outlined' pushed>TAG-3</Tag>
  </Row>
)
```

### Shape

`shape` свойство может быть `circle` (по умолчанию) либо `rounded`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Tag>CIRCLE</Tag>
    <Tag shape='rounded' pushed>ROUNDED</Tag>
  </Row>
)
```

### Tag с иконкой

Иконка может быть добавлена используя `icon` свойство, `iconPosition` может быть `left` (по умолчанию) либо `right`.

Tag можно сделать интерактивным используя `onPress` свойство.

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
const [showTags, setShowTags] = useState([true, true])
return (
  <Row>
    <Tag
      color='blue'
      style={{ marginRight: u(1) }}
      icon={faCoffee}
    >
       Non Interactive
    </Tag>
    {showTags[0] &&
      <Tag
        color='blue'
        style={{ marginRight: u(1) }}
        icon={faTimesCircle}
        onPress={() => setShowTags([false, showTags[1]])}
      >
        Interactive Left
      </Tag>
    }
    {showTags[1] &&
      <Tag
        color='blue'
        style={{ marginRight: u(1) }}
        icon={faTimesCircle}
        iconPosition='right'
        onPress={() => setShowTags([showTags[0], false])}
      >
        Interactive Right
      </Tag>
    }
  </Row>
)
```
