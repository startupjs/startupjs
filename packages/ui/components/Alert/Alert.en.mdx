# Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

```jsx
import { Alert } from '@startupjs/ui'
```

## Example
There are 4 types of Alert: `success`, `info`, `warning`, `error`. You can specify it by passing `variant` property.

```jsx example
return (
  <Div>
    <Alert
      variant='info'
      label='This is an info alert — check it out!'
    />
    <Br />
    <Alert
      variant='success'
      label='This is an success alert — check it out!'
    />
    <Br />
    <Alert
      variant='warning'
      label='This is an warning alert — check it out!'
    />
    <Br />
    <Alert
      variant='error'
      label='This is an error alert — check it out!'
    />
  </Div>
)
```

## Icons

The `icon` prop allows you to add an icon to the beginning of the alert component.
The `variant` prop defines the corresponding color for the icon.

```jsx example
return (
  <Div>
    <Alert
      icon={faCheck}
      variant='success'
      label='Success alert icon'
    />
    <Br />
    <Alert
      icon={faExclamationTriangle}
      variant='warning'
      label='Warning alert icon'
    />
  </Div>
)
```

## Actions

An alert can have an action (`onClose` prop). It is rendered after the message, at the end of the alert.

```jsx example
const [showAlert, setShowAlert] = useState(true)
return (
  <Div>
    {showAlert &&
      <Alert
        variant='warning'
        icon={faExclamationTriangle}
        label='Warning alert with actions'
        onClose={() => setShowAlert(false)}
      />
    }
    <Br />
    <Button disabled={showAlert} onPress={() => setShowAlert(true)}>
      Re-Open
    </Button>
  </Div>
)
```
