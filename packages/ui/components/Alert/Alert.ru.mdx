# Alert (оповещение)

Оповещение отображает короткое, важное сообщение таким образом, чтобы привлечь внимание пользователя, не прерывая его работу.

```js
import { Alert } from '@startupjs/ui'
```

## Пример

Существует 4 типа предупреждений, каждый с уникальным цветом и значком: `info`, `success`, `warning`, `error`. Вы можете указать тип, передав свойство `variant`.

```jsx example
return (
  <Div>
    <Alert
      variant='info'
      label='This is an info alert — check it out!'
    />
    <Br />
    <Alert
      variant='success'
      label='This is an success alert — check it out!'
    />
    <Br />
    <Alert
      variant='warning'
      label='This is an warning alert — check it out!'
    />
    <Br />
    <Alert
      variant='error'
      label='This is an error alert — check it out!'
    />
  </Div>
)
```

## Заголовок

Вы можете использовать свойство `title`, чтобы отобразить заголовок над содержимым.

```jsx example
return (
  <Div>
    <Alert
      variant='info'
      title='Info'
      label='This is an info alert — check it out!'
    />
    <Br />
    <Alert
      variant='success'

```

## Иконки

Свойство `icon` позволяет изменить иконку в начале компонента.
Свойство `variant` определяет соответствующий цвет и для иконки.

Установка свойства `icon` в значение `false` полностью удалит иконку.

```jsx example
return (
  <Div>
    <Alert
      icon={faFire}
      variant='error'
      label='Error alert with custom icon'
    />
    <Br />
    <Alert
      icon={false}
      variant='warning'
      label='Warning alert without icon'
    />
  </Div>
)
```

## Действия
Alert может иметь действие (`onClose` свойство), например кнопку закрытия или отмены. Действие рендерится после сообщения, в конце Alert.

Если передан колбек `onClose` и не задано свойство `ActionComponent`, отображается значок закрытия по умолчанию. Свойство `ActionComponent` можно использовать для предоставления альтернативного компонента, например, с помощью `Button`, `Tag` или `Icon`.

```jsx example
const [showFirstAlert, setShowFirstAlert] = useState(true)
const [showSecondAlert, setShowSecondAlert] = useState(true)

const renderCustomCloseButton = () => {
  return (
    <Button
      color='success'
      variant='flat'
      size='s'
      onPress={() => setShowSecondAlert(false)}
    >
      CLOSE
    </Button>
  )
}

const renderCustomActions = () => {
  return (
    <Row>
      <Button
        color='error'
        variant='text'
        icon={faTimes}
        size='s'
        onPress={() => {}}
      >
        Cancel
      </Button>
      <Button
        color='success'
        variant='text'
        icon={faCheck}
        size='s'
        onPress={() => {}}
      >
        Confirm
      </Button>
    </Row>
  )
}

return (
  <Div>
    {showFirstAlert && <Alert
      variant='warning'
      label='Warning alert with default actions'
      onClose={() => setShowFirstAlert(false)}
    />}
    <Br />
    <Button
      disabled={showFirstAlert}
      onPress={() => setShowFirstAlert(true)}
    >
      Re-Open
    </Button>
    <Br />
    {showSecondAlert && <Alert
      variant='success'
      label='Success alert with custom actions'
      ActionComponent={renderCustomCloseButton}
    />}
    <Br />
    <Button
      disabled={showSecondAlert}
      onPress={() => setShowSecondAlert(true)}
    >
      Re-Open
    </Button>
    <Br />
    <Alert
      variant='info'
      label='Info alert with two custom actions'
      ActionComponent={renderCustomActions}
    />
  </Div>
)
```
