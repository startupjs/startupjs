# Alert (оповещение)

Оповещение отображает короткое, важное сообщение таким образом, чтобы привлечь внимание пользователя, не прерывая его работу.

```jsx
import { Alert } from '@startupjs/ui'
```

## Пример
Существует 4 типа предупреждений: `success`, `info`, `warning`, `error`. Вы можете указать тип, передав свойство `variant`.

```jsx example
return (
  <Div>
    <Alert
      variant='info'
      label='This is an info alert — check it out!'
    />
    <Br />
    <Alert
      variant='success'
      label='This is an success alert — check it out!'
    />
    <Br />
    <Alert
      variant='warning'
      label='This is an warning alert — check it out!'
    />
    <Br />
    <Alert
      variant='error'
      label='This is an error alert — check it out!'
    />
  </Div>
)
```

## Иконки
Свойство `icon`позволяет добавить иконку в начало компонента оповещения.
Свойство `variant` определяет соответствующий цвет и для иконки.

```jsx example
return (
  <Div>
    <Alert
      icon={faCheck}
      variant='success'
      label='Success alert icon'
    />
    <Br />
    <Alert
      icon={faExclamationTriangle}
      variant='warning'
      label='Warning alert icon'
    />
  </Div>
)
```

## Действия

Оповещение может иметь действие (`onClose' свойство). Оно отображается после сообщения, в конце оповещения.

```jsx example
const [showAlert, setShowAlert] = useState(true)
return (
  <Div>
    {showAlert &&
      <Alert
        variant='warning'
        icon={faExclamationTriangle}
        label='Warning alert with actions'
        onClose={() => setShowAlert(false)}
      />
    }
    <Br />
    <Button disabled={showAlert} onPress={() => setShowAlert(true)}>
      Re-Open
    </Button>
  </Div>
)
```
