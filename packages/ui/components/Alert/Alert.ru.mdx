# Предупреждение

Предупреждение отображает короткое, важное сообщение таким образом, чтобы привлечь внимание пользователя

## Базовый пример

```jsx
import { Alert, Div, Br } from '@startupjs/ui'
```

```jsx example
return (
  <Div>
    <Alert
      label='Primary color alert'
    />
    <Br />
    <Alert
      color='success'
      label='Success alert'
    />
    <Br />
    <Alert
      color='warning'
      label='Warning alert'
    />
    <Br />
    <Alert
      color='error'
      label='Error alert'
    />
  </Div>
)
```

## Пример предупреждения с иконкой

```jsx
import { Alert, Div, Br } from '@startupjs/ui'
```

```jsx example
return (
  <Div>
    <Alert
      icon={faCheck}
      color='success'
      label='Success alert icon'
    />
    <Br />
    <Alert
      icon={faExclamationTriangle}
      color='warning'
      label='Warning alert icon'
    />
  </Div>
)
```

## Пример предупреждения c действием

```jsx
import { Alert, Div, Br, Button } from '@startupjs/ui'
```

```jsx example
const [showAlert, setShowAlert] = useState(true)
return (
  <Div>
    {showAlert &&
      <Alert
        color='primary'
        label='Alert with actions'
        onClose={() => setShowAlert(false)}
      />
    }
    <Br />
    <Button disabled={showAlert} onPress={() => setShowAlert(true)}>
      Re-Open
    </Button>
  </Div>
)
```
