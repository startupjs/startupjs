# Alert (оповещение)

Оповещение отображает короткое, важное сообщение таким образом, чтобы привлечь внимание пользователя, не прерывая его работу.

```jsx
import { Alert } from '@startupjs/ui'
```

## Пример

```jsx example
return (
  <Div>
    <Alert
      label='Primary color alert'
    />
    <Br />
    <Alert
      color='success'
      label='Success alert'
    />
    <Br />
    <Alert
      color='warning'
      label='Warning alert'
    />
    <Br />
    <Alert
      color='error'
      label='Error alert'
    />
  </Div>
)
```

## Иконки
Свойство `icon`позволяет добавить иконку в начало компонента оповещения.

```jsx example
return (
  <Div>
    <Alert
      icon={faCheck}
      iconStyleName='icon success'
      label='Success alert icon'
    />
    <Br />
    <Alert
      icon={faExclamationTriangle}
      iconStyleName='icon warning'
      label='Warning alert icon'
    />
  </Div>
)
```

## Действия

Оповещение может иметь действие (`onClose' свойство). Оно отображается после сообщения, в конце оповещения.

```jsx example
const [showAlert, setShowAlert] = useState(true)
return (
  <Div>
    {showAlert &&
      <Alert
        iconStyleName='icon'
        label='Alert with actions'
        onClose={() => setShowAlert(false)}
      />
    }
    <Br />
    <Button disabled={showAlert} onPress={() => setShowAlert(true)}>
      Re-Open
    </Button>
  </Div>
)
```
