# Br

`Br` is used to to insert a line break between text or element nodes. `Br` also lets the developer customize the number of line breaks needed.


## Example

Number of line breaks can be customized by specifying a number in `lines` prop. Default value of `lines` is `1`. 

```jsx
import { Br } from 'startupjs/ui'
```

```jsx example
const itemStyle = {
  height: u(5),
  justifyContent: 'center',
  alignItems: 'center'
}
const [value, setValue] = useState('1')
return (
  <Content>
    <TextInput
      label='Enter number of lines of space'
      value={value}
      onChangeText={(text) => setValue(text)}
    />
    <Br />
    <Div style={itemStyle} level='1'>
      <Span size='xl'>Item 1</Span>
    </Div>
    <Br lines={Number(value)} />
    <Div style={itemStyle} level='1'>
      <Span size='xl'>Item 2</Span>
    </Div>
  </Content>
)
```

## Half space

Setting the boolean `half` prop to true creates a space equivalent to a `lines` value of `0.5`. 

```jsx
import { Br } from 'startupjs/ui'
```

```jsx example
const itemStyle = {
  height: u(5),
  justifyContent: 'center',
  alignItems: 'center'
}
return (
  <Content>
    <TextInput
      disabled
      label='Enter number of lines of space'
      value='0.5'
    />
    <Br />
    <Div style={itemStyle} level='1'>
      <Span size='xl'>Item 1</Span>
    </Div>
    <Br half />
    <Div style={itemStyle} level='1'>
      <Span size='xl'>Item 2</Span>
    </Div>
  </Content>
)
```
