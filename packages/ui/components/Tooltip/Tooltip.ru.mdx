## Использование

- content: скрытая часть, может быть string или jsx
- children: видимая часть

## Тултип с нажимаемыми элементами

Если нужно обернуть нажимаемый элемент в тултип в мобильном приложении, свойство onPress должно быть в верхнем компоненте. Обратите внимание что тултип перезапишет свойства onLongPress и onPresOut.

- неверно

```jsx
<Tooltip
  content='Some content'
>
  <View>
    <Button
      onPress={onPressHandler}
    >
      Button...
    </Button>
  </View>
</Tooltip>
```

- верно

```jsx
<Tooltip
  content='Some content'
>
  <Button
    onPress={onPressHandler}
  >
    Button...
  </Button>
</Tooltip>
```

## Пример с кнопкой

```jsx example
const [pressed, setPressed] = useState('')

return (
  <View>
    <View 
      style={{
        width: 100,
        marginBottom: 16
      }}
    >
      <Tooltip
        content='Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.'
      >
        <Button
          onPress={() => setPressed('onPress')}
          onLongPress={() => setPressed('onLongPress')}
        >
          Button...
        </Button>
      </Tooltip>
    </View>
    {!!pressed && (
      <Alert
        variant='success'
        label={`${pressed} event`}
        onClose={() => setPressed()}
      />
    )}
  </View>
)
```
