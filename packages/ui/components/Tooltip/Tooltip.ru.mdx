# Tooltip

```jsx
import { Tooltip } from '@startupjs/ui'
```

## Использование

- content: скрытая часть, может быть string или jsx
- children: видимая часть

## Пример с текстом

```jsx example
const wrapperStyle = { backgroundColor: '#dddddd', height: 40, borderRadius: 6, width: 100, alignItems: 'center', justifyContent: 'center' }

const content = 'Задача организации, в особенности же постоянное информационно-пропагандистское обеспечение нашей деятельности требуют определения и уточнения новых предложений.'

return (
  <Row>
    <Tooltip content={content} style={{ marginRight: 8 }}>
      <Div style={wrapperStyle}>
        <Span>Caption 1</Span>
      </Div>
    </Tooltip>
    <Tooltip content={content} style={{ marginRight: 8 }}>
      <Div style={wrapperStyle}>
        <Span>Caption 2</Span>
      </Div>
    </Tooltip>
    <Tooltip content={content} style={{ marginRight: 8 }}>
       <Div style={wrapperStyle}>
        <Span>Caption 3</Span>
      </Div>
    </Tooltip>
  </Row>
)
```

## Тултип с нажимаемыми элементами

Если нужно обернуть нажимаемый элемент в тултип в мобильном приложении, свойство `onPress` должно быть в верхнем компоненте.

- неверно

```jsx
<Tooltip content='Some content'>
  <View>
    <Button onPress={onPressHandler}>Button...</Button>
  </View>
</Tooltip>
```

- верно

```jsx
<Tooltip content='Some content'>
  <Button onPress={onPressHandler}>Button...</Button>
</Tooltip>
```

## Пример с кнопкой

```jsx example
const [pressed, setPressed] = useState('')

const content = 'Задача организации, в особенности же постоянное информационно-пропагандистское обеспечение нашей деятельности требуют определения и уточнения новых предложений.'

return (
  <>
    <Tooltip content={content}>
      <Button onPress={()=> setPressed('onPress')}>Click me</Button>
    </Tooltip>
    {!!pressed && (
      <Div style={{ marginTop: 8 }}>
        <Alert
          variant='success'
          label={`${pressed} event`}
          onClose={()=> setPressed()}
        />
      </Div>
    )}
  </>
)
```
