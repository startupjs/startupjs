# Loader (лоадер)

Лоадеры используются для отображения времени ожидания.

```jsx
import { Loader } from '@startupjs/ui'
```

## Пример

```jsx example
return (
  <Loader />
)
```

## Размеры

Лоадеры могут быть двух размеров - `s` и `m`.

```jsx example
return (
  <Row align='around'>
    <Loader size='s' />
    <Loader size='m' />
  </Row>
)
```

## Цвета

Цвет можно изменить, передав шестнадцатеричный цвет или имя цвета из цветов `ui` конфига в свойство `color`.

```jsx example
return (
  <Row align='around'>
    <Loader color='#2962FF' />
    <Loader color='#F5BE59' />
    <Loader color='#00AED6' />
    <Loader color='#A6D800' />
  </Row>
)
```
