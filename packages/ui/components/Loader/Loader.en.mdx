# Loader

Loaders are used to show a wait time.

## Example

```jsx
import { Loader } from '@startupjs/ui'
```

```jsx example
return (
  <Loader />
)
```

## Sizes

Loaders can be of two sizes - `s` and `m`.

```jsx
import { Loader } from '@startupjs/ui'
```

```jsx example
return (
  <Row align='around'>
    <Loader size='s' />
    <Loader size='m' />
  </Row>
)
```

## Colors

```jsx
import { Loader } from '@startupjs/ui'
```

```jsx example
return (
  <Row align='around'>
    <Loader color='#2962FF' />
    <Loader color='#F5BE59' />
    <Loader color='#00AED6' />
    <Loader color='#A6D800' />
  </Row>
)
```
