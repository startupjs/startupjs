# Typography

A set of components to make text clear and easy to read to the end user. The base component of typography is `Span`, which means that all other components use it in different variations for example headers.

```jsx
import { Span } from '@startupjs/ui'
import { H1, H2, H3, H4, H5, H6, Content } from '@startupjs/ui'
```

## Span

```jsx example
  <Span>Lorem ipsum dolor sit amet, consectetur adipisicing elit</Span>
```

## Sizes

To change text size use `font(size)` global mixin. Possible values of `size` (`xs`, `s`, `m`, `l`, `xl`, `xxl`, `xxxl`, `xxxxl`, `xxxxxl`). `m` is default `Span` size.

In `.styl` file
```stylus
.text
  font(xxl)
```

```jsx example
  <Span styleName='text'>Text size - xxl.</Span>
```

## Variants

The `variant` property means **"make text as"** and it uses `text(key)` global mixin under the hood. Possible values of `key` (`h1`, `h2`, `h3`, `h4`, `h5`, `h6`, `description`). **We do not recommend using mixin unnecessarily instead of `variant` property**.

```jsx example
return (
  <Content width='full'>
    <Span variant='h1'>
      Text size - xxxxxl.
    </Span>
    <Br />
    <Span variant='h2'>
      Text size - xxxxl.
    </Span>
    <Br />
    <Span variant='h3'>
      Text size - xxxl.
    </Span>
    <Br />
    <Span variant='h4'>
      Text size - xxl.
    </Span>
    <Br />
    <Span variant='h5'>
      Text size - xl.
    </Span>
    <Br />
    <Span variant='h6'>
      Text size - l.
    </Span>
    <Br />
    <Span variant='description'>
      Text size - m.
    </Span>
  </Content>
)
```

## Headers

H* components are used for heading or subheadings of a section or a page.

```jsx example
return (
  <Content>
    <H1>H1 Heading</H1>
    <Br half />
    <H2>H2 Heading</H2>
    <Br half />
    <H3>H3 Heading</H3>
    <Br half />
    <H4>H4 Heading</H4>
    <Br half />
    <H5>H5 Heading</H5>
    <Br half />
    <H6>H6 Heading</H6>
  </Content>
)
```

## Bold text

To make text bold pass the property `bold` to component.

```jsx example
return (
  <Content width='full'>
    <Span bold size='xxl'>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit.
    </Span>
    <H4 bold>H4 Heading</H4>
  </Content>
)
```

## Italic text

To make text italic pass the property `italic` to component.

```jsx example
return (
  <Content width='full'>
    <Span italic size='xxl'>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit.
    </Span>
    <H4 italic>H4 Heading</H4>
  </Content>
)
```

## Insignificant text (Span only)

To reduce the significance of the text pass the property `description` to component.

```jsx example
return (
  <Span description>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quos blanditiis tenetur unde suscipit, quam beatae rerum inventore consectetur, neque doloribus, cupiditate numquam dignissimos laborum fugiat deleniti? Eum quasi quidem quibusdam.
  </Span>
)
```
