# Typography (типография)

Набор компонентов, чтобы сделать текст понятным и легко читаемым для конечного пользователя. Базовый компонент типографики - это `Span`, что означает, что все остальные компоненты используют его в различных вариациях, например заголовки.

```jsx
import { Span } from '@startupjs/ui'
import { H1, H2, H3, H4, H5, H6, Content } from '@startupjs/ui'
```

## Span (спан)

```jsx example
  <Span>Lorem ipsum dolor sit amet, consectetur adipisicing elit</Span>
```

## Размеры

Для изменения размера текста используйте глобальный миксин `font(size)`. Возможные значения `size` (`xs`, `s`, `m`, `l`, `xl`, `xxl`, `xxxl`, `xxxxl`, `xxxxxl`). `m` - это стандартный размер `Span`.

В `.styl` файле
```stylus
.text
  font(xxl)
```

```jsx example
  <Span styleName='text'>Text size - xxl.</Span>
```

## Разновидности

Свойство `variant` означает **"сделать текст как"** и использует `text(key)` глобальный миксин под капотом. Возможные значения `key` (`default`, `h1`, `h2`, `h3`, `h4`, `h5`, `h6`, `description`). **Мы не рекомендуем использовать миксин без необходимости вместо свойства `variant`**.

```jsx example
return (
  <Content width='full'>
    <Span>
      Размер текста - m (стандартный).
    </Span>
    <Br />
    <Span variant='h1'>
      Размер текста - xxxxxl.
    </Span>
    <Br />
    <Span variant='h2'>
      Размер текста - xxxxl.
    </Span>
    <Br />
    <Span variant='h3'>
      Размер текста - xxxl.
    </Span>
    <Br />
    <Span variant='h4'>
      Размер текста - xxl.
    </Span>
    <Br />
    <Span variant='h5'>
      Размер текста - xl.
    </Span>
    <Br />
    <Span variant='h6'>
      Размер текста - l.
    </Span>
    <Br />
    <Span variant='description'>
      Размер текста - m.
    </Span>
  </Content>
)
```




## Заголовки

H* комопненты используются для заголовок или подзаголовков раздела или страницы.

```jsx example
return (
  <Content>
    <H1>H1 Heading</H1>
    <Br half />
    <H2>H2 Heading</H2>
    <Br half />
    <H3>H3 Heading</H3>
    <Br half />
    <H4>H4 Heading</H4>
    <Br half />
    <H5>H5 Heading</H5>
    <Br half />
    <H6>H6 Heading</H6>
  </Content>
)
```

## Жирный текст

Чтобы сделать текст жирным, передайте компоненту свойство `bold`.

```jsx example
return (
  <Content>
    <Span bold size='xxl'>H4 Heading using Span</Span>
    <H4 bold>H4 Heading</H4>
  </Content>
)
```

## Курсив

Чтобы сделать текст курсивом, передайте компоненту свойство `italic`.

```jsx example
return (
  <Content width='full'>
    <Span italic size='xxl'>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit.
    </Span>
    <H4 italic>H4 Heading</H4>
  </Content>
)
```

## Малозначимый текст (только для Span)

Чтобы уменьшить значимость текста, передайте компоненту свойство `description`.

```jsx example
return (
  <Span description>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quos blanditiis tenetur unde suscipit, quam beatae rerum inventore consectetur, neque doloribus, cupiditate numquam dignissimos laborum fugiat deleniti? Eum quasi quidem quibusdam.
  </Span>
)
```
