# DrawerSidebar

Drawer is typically used for nagivation. It is initially not visible on the screen, but it can be pulled in from the edge of the window.

## Example

```jsx
import { DrawerSidebar, Menu, Content, Row } from '@startupjs/ui'
```

```jsx example
const PATH = '_session.ExampleLeftDrawerSidebar'
const $opened = useModel(PATH)
useEffect(() => {
  $opened.set(true)
}, [])
return (
  <Div style={{overflow: 'hidden'}}>
    <DrawerSidebar
      path={PATH}
      renderContent={() => (
        <Menu>
          <Menu.Item>Nav-1</Menu.Item>
          <Menu.Item>Nav-2</Menu.Item>
          <Menu.Item>Nav-3</Menu.Item>
          <Menu.Item>Nav-4</Menu.Item>
        </Menu>
      )}
    >
      <Content
        padding
        width='full'
        style={{ height: 200, backgroundColor: 'white' }}
      >
        <Row>
          <Button onPress={() => $opened.set(true)}>Open</Button>
        </Row>
      </Content>
    </DrawerSidebar>
  </Div>
)
```

## Position

The `position` property specifies the side of the window from which the drawer will slide in. Possible values of `position` property are `left` (default) and `right`.

```jsx
import { DrawerSidebar, Menu, Content, Row } from '@startupjs/ui'
```

```jsx example
const PATH = '_session.ExampleRightDrawerSidebar'
const $opened = useModel(PATH)
useEffect(() => {
  $opened.set(true)

    </DrawerSidebar>
  </Div>

```

## Locked Drawer

Specifying `forceClosed` property will keep the drawer closed and it won't respond to gestures.

```jsx example
const PATH = '_session.ExampleLockedDrawerSidebar'
const $opened = useModel(PATH)
useEffect(() => {
  $opened.set(true)
}, [])
return (
  <Div style={{overflow: 'hidden'}}>
    <DrawerSidebar
      path={PATH}
      renderContent={() => (
        <Menu>
          <Menu.Item>Nav-1</Menu.Item>
          <Menu.Item>Nav-2</Menu.Item>
          <Menu.Item>Nav-3</Menu.Item>
          <Menu.Item>Nav-4</Menu.Item>
        </Menu>
      )}
      forceClosed
    >
      <Content
        padding
        width='full'
        style={{ height: 200, backgroundColor: 'white' }}
      >
        <Row>
          <Button onPress={() => $opened.set(true)}>Open</Button>
        </Row>
      </Content>
    </DrawerSidebar>
  </Div>
)
```
