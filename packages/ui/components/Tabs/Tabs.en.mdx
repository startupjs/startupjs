# Tabs

Tabs display a list of cards and components for navigating through them.

```jsx
import { Tabs } from '@startupjs/ui'
```

## Example

```jsx example
const [value, setValue] = useState(0)

return (
  <Div>
  <Tabs
    value={value}
    onChange={setValue}
  >
    <Tabs.Item
      title='1 tab'
    >
      <ExampleTab bgColor='blue' />
    </Tabs.Item>
    <Tabs.Item
      value='qwe'
      title='2 tab'
    >
      <ExampleTab bgColor='pink' />
    </Tabs.Item>
    <Tabs.Item
      title='3 tab'
    >
      <ExampleTab bgColor='aqua' />
    </Tabs.Item>
  </Tabs>
  </Div>
)
```
