# Tabs

Tabs display a list of cards and components for navigating through them.

```jsx
import { Tabs } from '@startupjs/ui'
```

## Example

```jsx example
const [value, $value] = useValue('Second')

return (
  <Div>
  <Tabs
    $value={$value}
  >
    <Tabs.Item
      title='1 tab'
    >
      <ExampleTab bgColor='blue' />
    </Tabs.Item>
    <Tabs.Item
      value='Second'
      title='2 tab'
    >
      <ExampleTab bgColor='pink' />
    </Tabs.Item>
    <Tabs.Item
      title='3 tab'
    >
      <ExampleTab bgColor='aqua' />
    </Tabs.Item>
  </Tabs>
  </Div>
)
```
