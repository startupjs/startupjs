# User

Inherits [Div props](/docs/en/Div).

User is used to display information about users. It uses [Avatar](/docs/en/Avatar) component to display user avatar.

## Example

```jsx
import { User, Content, Br } from '@startupjs/ui'
```

```jsx example
return (
  <Content>
    <User
      avatarUrl='/img/avatar1.jpeg'
      name='John Doe'
      description='The cat is a domestic species of small carnivorous mammal'
    />
    <Br />
    <User
      name='John Doe'
      description='The cat is a domestic species of small carnivorous mammal'
    />
  </Content>
)
```

## Sizes

Size is `m` by default. It can be changed by changing `size` prop. Possible values of property can be found in the `Sandbox` section at the bottom of page.

```jsx
import { User, Div, Br } from '@startupjs/ui'
```

```jsx example
return (
  <Div>
    <User
      size='xs'
      avatarUrl='/img/avatar1.jpeg'
      name='John Doe'
      description='xs size'
    />
    <Br />
    <User
      size='s'
      avatarUrl='/img/avatar2.jpeg'
      name='John Doe'
      description='s size'
    />
    <Br />
    <User
```

## User status

To display user online status on avatar you can pass string `online` or `away` to property `status`.

```jsx
import { User, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <User
      status='online'
      avatarUrl='/img/avatar1.jpeg'
      name='John Doe'
      description='Online description'
        style={{ marginRight: u(2) }}
    />
    <User
      status='away'
      avatarUrl='/img/avatar2.jpeg'
      name='John Doe'
      description='Offline description'
      />
  </Row>
)
```

## User avatar position

Avatar position can be modified by passing string `left` (default) or `right` to `avatarPosition` property.

```jsx
import { User } from '@startupjs/ui'
```

```jsx example
return (
  <User
    avatarPosition='right'
    avatarUrl='/img/avatar1.jpeg'
    name='John Doe'
    description='The cat is a domestic species of small carnivorous mammal'
  />
)
```

## Actions

Passed handler to `onPress` property will be called when the user taps the component.

```jsx
import { User, Content } from '@startupjs/ui'
```

```jsx example
const [clicked, setClicked] = useState(0)
return (
  <Content>
    <User
      avatarUrl='/img/avatar1.jpeg'
      name='John Doe'
      description='The cat is a domestic species of small carnivorous mammal'
      onPress={() => setClicked(clicked + 1)}
    />
    <Br />
    <Span>
      {clicked ? `User clicked ${clicked} ${clicked === 1 ? 'time' : 'times'}` : 'Click on user'}
    </Span>
  </Content>
)
```
