# Typography (типография)

Набор компонентов, чтобы сделать текст понятным и легко читаемым для конечного пользователя. Базовый компонент типографики - это `Span`, что означает, что все остальные компоненты используют его в различных вариациях, например заголовки.

```jsx
import { Span, H1, H2, H3, H4, H5, H6 } from '@startupjs/ui'
```

## Span (спан)

```jsx example
  <Span>Lorem ipsum dolor sit amet, consectetur adipisicing elit</Span>
```

## Размеры

Для изменения размера текста используйте глобальный миксин `font(size)`. Возможные значения `size` (`xs`, `s`, `m`, `l`, `xl`, `xxl`, `xxxl`, `xxxxl`, `xxxxxl`). `m` - это стандартный размер `Span`.

В `.styl` файле
```stylus
.text
  font(xxl)
```

```jsx example
  <Span styleName='text'>Text size - xxl.</Span>
```

## Разновидности

Свойство `variant` означает **"сделать текст как"** и использует `text(key)` глобальный миксин под капотом. Возможные значения `key` (`default`, `h1`, `h2`, `h3`, `h4`, `h5`, `h6`, `description`). **Мы не рекомендуем использовать миксин без необходимости вместо свойства `variant`**.

```jsx example
return (
  <Div>
    <Span>
      Default (размер - m, цвет - $UI.colors.mainText)
    </Span>
    <Br />
    <Br />
    <Span variant='description'>
      Description (используйте для уменьшения значимости текста, размер - m, цвет - $UI.colors.secondaryText)
    </Span>
    <Br />
    <Br />
    <Span variant='h1'>
      H1 (размер - xxxxxl, цвет - $UI.colors.mainText)
    </Span>
    <Br />
    <Br />
    <Span variant='h2'>
      H2 (размер - xxxxl, цвет - $UI.colors.mainText)
    </Span>
    <Br />
    <Br />
    <Span variant='h3'>
      H3 (размер - xxxl, цвет - $UI.colors.mainText)
    </Span>
    <Br />
    <Br />
    <Span variant='h4'>
      H4 (размер - xxl, цвет - $UI.colors.mainText)
    </Span>
    <Br />
    <Br />
    <Span variant='h5'>
      H5 (размер - xl, цвет - $UI.colors.mainText)
    </Span>
    <Br />
    <Br />
    <Span variant='h6'>
      H6 (размер - l, цвет - $UI.colors.mainText)
    </Span>
  </Div>
)
```

## Заголовки

`H*` комопненты используются для заголовок или подзаголовков раздела или страницы. Основное отличие от компонента `Span` с свойством `variant` заключается в том, что `H*` вставляют семантические теги в DOM на вебе.

```jsx example
return (
  <Div>
    <H1>H1 Heading</H1>
    <Br half />
    <H2>H2 Heading</H2>
    <Br half />
    <H3>H3 Heading</H3>
    <Br half />
    <H4>H4 Heading</H4>
    <Br half />
    <H5>H5 Heading</H5>
    <Br half />
    <H6>H6 Heading</H6>
  </Div>
)
```

## Жирный текст

Чтобы сделать текст жирным, передайте компоненту свойство `bold`.

```jsx example
return (
  <Div>
    <Span bold>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</Span>
    <H4 bold>H4 Heading</H4>
  </Div>
)
```

## Курсив

Чтобы сделать текст курсивом, передайте компоненту свойство `italic`.

```jsx example
return (
  <Div>
    <Span italic>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</Span>
    <H4 italic>H4 Heading</H4>
  </Div>
)
```
