# Table

The Table component allows to arrange data in rows and columns. The table can contain a header and a body, which are specified using the Thead and Tbody components. Any table consists of rows and cells, which are specified using the Tr and Td components.

```jsx
import { Table, Thead, Tbody, Tr, Td } from '@startupjs/ui'
```

## Example

```jsx example
const tableStyle = {
  backgroundColor: '#fff',
}

return (
  <Table style={tableStyle}>
    <Thead>
      <Tr>
        <Td>
          <Span bold>First</Span>
        </Td>
        <Td>
          <Span bold>Second</Span>
        </Td>
        <Td>
          <Span bold>Third</Span>
        </Td>
      </Tr>
    </Thead>
    <Tbody>
      <Tr>
        <Td>
          <Span italic>First 1</Span>
        </Td>
        <Td>Second 1</Td>
        <Td>Third 1</Td>
      </Tr>
      <Tr>
        <Td>
          <Span italic>First 2</Span>
        </Td>
        <Td>Second 2</Td>
        <Td>Third 2</Td>
      </Tr>
    </Tbody>
  </Table>
)
```
