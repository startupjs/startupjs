# Dropdown
Всплывающие меню. Адаптируемое в зависимости от расширения.

```jsx
import { Dropdown } from '@startupjs/ui'
```

## Инициализация

Перед использованием нужно настроить [Portal](/docs/components/Portal)

## Простой пример
- value: активное значение
- options: массив объектов для опций, каждая опция имеет значение (value) и метку (label)
- placeholder: заголовок поля
- onChange: калбэк функция, вызывается после выбора значения, принимает выбранное значение первым параметром

```jsx example
const [sort, setSort] = useState({})

const options = [
  { value: 'popular', label: 'Popular' },
  { value: 'brand', label: 'Brand' },
  { value: 'name', label: 'Name' }
]

return pug`
  Dropdown(
    value=sort
    options=options
    placeholder="Sort by"
    onChange=v=> setSort(v)
  )
    Row
      Span.title(bold) Sort by:
      if sort
        Span= sort.label
`

styl`
  .title
    margin-right .5u
`
```

## Расширеные значения options
```jsx example
const [sort, setSort] = useState('')

const options = [
  { value: 'edit', label: 'Edit', icon: faPencilAlt },
  { value: 'delete', label: 'Delete', icon: faTrashAlt }
]

return pug`
  Dropdown(
    value=sort
    options=options
    onChange=v=> setSort(v)
  )
    Button Open
`
```
