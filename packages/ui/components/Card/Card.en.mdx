# Card

Card is used to display content about a single subject.
Card inherits props of [`Div`](/docs/en/docs/Div) component.

## Basic Example

```jsx
import { Card, Span, Br, H4 } from '@startupjs/ui'
```

```jsx example
return (
  <Card onPress={() => console.log('You clicked the card')}>
    <H4>Card header</H4>
    <Br />
    <Span size='xl'>Card body or description</Span>
  </Card>
)
``` 

## Card Level Example

Card `level` prop determines how elevated the card will be.
Its only effective when the `variant` prop is set to `elevated`.

```jsx
import { Card, Span, Br, H4 } from '@startupjs/ui'
```

```jsx example
return (
  <Div>
    <Card>
      <H4>Card Level - 1</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
    <Br />
    <Card level={2}>
      <H4>Card Level - 2</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
    <Br />
    <Card level={3}>
      <H4>Card Level - 3</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
    <Br />
    <Card level={4}>
      <H4>Card Level - 3</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
  </Div>
  
)
``` 

## Card Variant Example

`variant` can be either `elevated` or `outlined`. It is set to `elevated` by default.

```jsx
import { Card, Span, Br, H4, Icon } from '@startupjs/ui'
```

```jsx example
return (
  <Div>
    <Card variant='outlined'>
      <H4>Card header</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
    <Br />
    <Card>
      <H4>Card header</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>    
  </Div>
)
```

## Intergration with other components example

```jsx
import { Image } from 'react-native'
import { Card, Span, Br, H4, Collapse, Row, Button } from '@startupjs/ui'
import { faShareAlt } from '@fortawesome/free-solid-svg-icons'
```

```jsx example
const style = {
  width: u(50)
}
const imageStyle = {
  width: u(50),
  height: u(40),
  resizeMode: 'cover',
  marginLeft: u(-2),
  marginTop: u(-2)
}
const [collapsed, setCollapsed] = useState(false)
return (
  <Card
    style={style}
  >
    <Image
      source={{ uri: '/img/cat.webp' }}
      style={imageStyle}
    />
    <Br />
    <H4>Cat</H4>
    <Br />
    <Span size='xl'>
      The cat (Felis catus) is a domestic species of small carnivorous mammal.
    </Span>
    <Br />
    <Row align='between'>
      <Button
        onPress={() => setCollapsed(!collapsed)}
        variant='flat'      
      >
        Learn more
      </Button>
      <Button
        icon={faShareAlt}
        onPress={() => null}
      >
        Share
      </Button>
    </Row>
    <Br />
    <Collapse open={collapsed} variant='pure'>
      <Collapse.Content>
        It is the only domesticated species in the family Felidae and is often referred to as the domestic cat to distinguish it from the wild members of the family. A cat can either be a house cat, a farm cat or a feral cat; the latter ranges freely and avoids human contact. Domestic cats are valued by humans for companionship and their ability to hunt pests such as rodents. About 60 cat breeds are recognized by various cat registries. (source: wiki)
      </Collapse.Content>
    </Collapse>
  </Card>
)
```
