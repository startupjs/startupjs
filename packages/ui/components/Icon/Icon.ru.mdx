# Icon (иконка)

Иконки - это визуальные символы, используемые для представления идей, объектов или действий. Они передают посыл с первого взгляда, обеспечивают интерактивность и привлекают внимание к важной информации.

```jsx
import { Icon } from '@startupjs/ui'
```

## Fontawesome иконки

```jsx
import {
  faHouseUser,
  faTable,
  faShareAlt
} from '@fortawesome/free-solid-svg-icons'
```

```jsx example
return (
  <Row align='around'>
    <Icon icon={faShareAlt} />
    <Icon icon={faHouseUser} />
    <Icon icon={faTable} />
  </Row>
)
```

## Размеры

Размер можно изменить с помощью свойства `size`. Значение по умолчанию `m`. Предопределенные значения размеров иконок - `xs`,` s`, `m`,` l`, `xl`, ` xxl`. Размер также может быть сколь угодно увеличен или уменьшен путем передачи числа в свойство `size`.

```jsx
import {
  faHouseUser,
  faTable,
  faShareAlt
} from '@fortawesome/free-solid-svg-icons'
```

```jsx example
const [numSize, setNumSize] = useState('80')
return (
  <Content>
    <Row align='around'>
      <Icon size='xs' icon={faShareAlt} />
      <Icon size='s' icon={faHouseUser} />
      <Icon icon={faTable} />
      <Icon size='l' icon={faShareAlt} />
      <Icon size='xl' icon={faHouseUser} />
      <Icon size='xxl' icon={faTable} />
    </Row>
    <Br />
    <TextInput
      label='Type in number to change icon size'
      value={numSize}
      onChangeText={value => setNumSize(value)}
    />
    <Br />
    <Icon size={+numSize} icon={faTable} />
  </Content>
)
```

## Цвета

Цвет можно изменить, передав шестнадцатеричный цвет или имя цвета из цветов `ui` конфига в свойство `color`.

```jsx
import {
  faHouseUser,
  faTable,
  faShareAlt
} from '@fortawesome/free-solid-svg-icons'
```

```jsx example
return (
  <Row align='around'>
    <Icon color='#24B47E' icon={faShareAlt} />
    <Icon color='#6772E5' icon={faHouseUser} />
    <Icon color='primary' icon={faTable} />
  </Row>
)
```

## Кастомная иконка

Чтобы использовать свою собственную иконку необходимо передать валидную `svg` иконку в свойство `icon`

```jsx
import CustomIconExample from './CustomIconExample.svg'
```

```jsx example
return (
  <Content>
    <Icon
      size={50}
      color='#F5BE59'
      icon={CustomIconExample}
    />
  </Content>
)
```
