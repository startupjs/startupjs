# Icon

Icons are visual symbols used to represent ideas, objects, or actions. They communicate messages at a glance, afford interactivity, and draw attention to important information.

```jsx
import { Icon } from '@startupjs/ui'
```

## Fontawesome icons

Component uses `@fortawesome/react-native-fontawesome` library to implement icons.

```jsx
import {
  faHouseUser,
  faTable,
  faShareAlt
} from '@fortawesome/free-solid-svg-icons'
```

```jsx example
return (
  <Row align='around'>
    <Icon icon={faShareAlt} />
    <Icon icon={faHouseUser} />
    <Icon icon={faTable} />
  </Row>
)
```

## Sizes

Size can be modified using the `size` prop. Default value is `m`. Pre defined icon size values are `xs`, `s`, `m`, `l`, `xl`, `xxl`. Size can also be arbitarily increased or decreased by providing a number to `size` prop.

```jsx
import {
  faHouseUser,
  faTable,
  faShareAlt
} from '@fortawesome/free-solid-svg-icons'
```

```jsx example
const [numSize, setNumSize] = useState('80')
return (
  <Content>
    <Row align='around'>
      <Icon size='xs' icon={faShareAlt} />
      <Icon size='s' icon={faHouseUser} />
      <Icon icon={faTable} />
      <Icon size='l' icon={faShareAlt} />
      <Icon size='xl' icon={faHouseUser} />
      <Icon size='xxl' icon={faTable} />
    </Row>
    <Br />
    <TextInput
      label='Type in number to change icon size'
      value={numSize}
      onChangeText={value => setNumSize(value)}
    />
    <Br />
    <Icon size={+numSize} icon={faTable} />
  </Content>
)
```

## Colors

Color can be changed by providing hex color or name of the color from `ui` config colors to `color` property.

```jsx
import {
  faHouseUser,
  faTable,
  faShareAlt
} from '@fortawesome/free-solid-svg-icons'
```

```jsx example
return (
  <Row align='around'>
    <Icon color='#24B47E' icon={faShareAlt} />
    <Icon color='#6772E5' icon={faHouseUser} />
    <Icon color='primary' icon={faTable} />
  </Row>
)
```

## Custom Icon

To use your custom icon you must provide a valid `svg` icon to `icon` property

```jsx
import CustomIconExample from './CustomIconExample.svg'
```

```jsx example
return (
  <Icon
    size={50}
    color='#F5BE59'
    icon={CustomIconExample}
  />
)
```
