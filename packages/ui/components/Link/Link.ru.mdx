# Link (ссылка)

Наследует [Span props](/docs/ru/typography).

Ссылка позволяет перемещаться по приложению и настраивать якоря.

```jsx
import { Link } from '@startupjs/ui'
```

## Пример

```jsx example
return (
  <Span>
    Наследует&nbsp;
    <Link to='/docs/ru/typography'>
      Span props
    </Link>
  </Span>
)
```

## Цвета

Чтобы изменить цвет ссылки передайте свойство `color`.

```jsx example
return (
  <Row>
    <Span>Inherits&nbsp;</Span>
    <Link block to='/docs/ru/typography' color='primary'>
      <Span>Span props</Span>
    </Link>
  </Row>
)
```
