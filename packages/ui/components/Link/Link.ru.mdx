# Link (ссылка)

Наследует [Span props](/docs/ru/typography).

Ссылка позволяет перемещаться по приложению и настраивать якоря.

```jsx
import { Link } from '@startupjs/ui'
```

## Пример

```jsx example
return (
  <Span>
    Наследует&nbsp;
    <Link to='/docs/ru/typography'>
      Span props
    </Link>
  </Span>
)
```

## Кастомная разметка

Свойство `block` используется для добавления пользовательского контента в компонент.

```jsx example
return (
  <Span>
    Inherits&nbsp;
    <Link block to='/docs/en/typography'>
      <Span>Span props</Span>
    </Link>
  </Span>
)
```

## Цвета

Чтобы изменить цвет ссылки передайте свойство `color` (не работает со свойством `block`).

```jsx example
return (
  <Row>
    <Span>Inherits&nbsp;</Span>
    <Link block to='/docs/ru/typography' color='primary'>
      <Span>Span props</Span>
    </Link>
  </Row>
)
```

## Действия

Передавая `onPress` обработчик в компонент, вы можете выполнить некоторую логику перед перенаправлением.

```jsx example
function onPress (event) {
  console.log('Go to alert docs')
}
return (
  <Link
    color='primary'
    to='/docs/en/components/Alert'
    onPress={onPress}
  >
    Alert docs
  </Link>
)
```
