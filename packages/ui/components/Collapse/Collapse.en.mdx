# Collapse

Collapse allows to toggle the visibility of content.

## Example

```jsx
import { Collapse } from '@startupjs/ui'
```

```jsx example
const [open, setOpen] = useState(false)
return (
  <Collapse title='Simple collapse' open={open} onChange={() => setOpen(!open)}>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. ## Pure collapse
```

Pure collapse does not contain any styles (background, shadow, paddings, etc.). It is useful when `Collapse` is used as a children of other components like `Card`. To make it pure, pass the string `pure` to the `variant` property.

```jsx
import { Collapse } from '@startupjs/ui'
```

```jsx example
const [open, setOpen] = useState(false)
return (
  <Collapse
    variant='pure'
    open={open}
    onChange={() => setOpen(!open)}
    title='Pure collapse'
  >
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.   </Collapse>
)
```

## Advanced usage

Collapse consists of two parts - `Header` and `Content`. For example the parts can be used to add custom markup. Instead of `title` using `Header` and instead of `children` using `Content.` They can be used separately. Each part take value of `variant` prop from parent `Collapse` which can be overridden by passing `variant` to itself.

```jsx
import { Collapse } from '@startupjs/ui'
```

```jsx example
const [open, setOpen] = useState(false)
return (
  <Collapse open={open} onChange={() => setOpen(!open)}>
    <Collapse.Header iconPosition='right'>Advanced collapse</Collapse.Header>
    <Collapse.Content>
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum
    </Collapse.Content>
  </Collapse>
)
```

## Accordion

Using the `Collapse` component, you can create behavior like an accordion.

```jsx
import { Collapse, Div, LineSeparator } from '@startupjs/ui'
```

```jsx example
const [expand, setExpand] = useState()
const handleChange = (panel) => (shouldExpand) => {
  setExpand(shouldExpand ? panel : null)
}
return (
  <Div>
    <Collapse
      open={expand === 'open1'}
      title='Accordion Header - 1'
      onChange={handleChange('open1')}
    >
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco.
    </Collapse>
    <LineSeparator />
    <Collapse
      open={expand === 'open2'}
      title='Accordion Header - 2'
      onChange={handleChange('open2')}
    >
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco.
    </Collapse>
    <LineSeparator />
    <Collapse
      open={expand === 'open3'}
      title='Accordion Header - 3'
      onChange={handleChange('open3')}
    >
      Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco.
    </Collapse>
  </Div>
)
```
