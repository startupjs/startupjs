# Avatar

Avatar можно использовать для отображения изображений, значков, букв.

## Базовый пример

```jsx
import { Avatar } from '@startupjs/ui'
```

```jsx example
return (
  <Avatar
    url='/img/avatar1.jpeg'
  />
)
```

## Пример со статусом

```jsx
import { Avatar, Row, Div } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Div>
      <Avatar
        status='online'
        url='/img/avatar1.jpeg'
      />
    </Div>
    <Div pushed>
      <Avatar
        status='away'
        url='/img/avatar2.jpeg'
      />
    </Div>
  </Row>
)
```

## Размеры

Avatar может быть разных размеров.

```jsx

```


## Резервная строка

Когда свойство `url` отсутствует или он не может быть загружен, то отображаются инициалы `резервной строки`.
В приведенном ниже примере `резервная строка` 'John Doe', поэтому будет отображаться 'JD'. По умолчанию `резервная строка` - `?`

```jsx
import { Avatar, Row, Div } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Div>
      <Avatar
        url='/img/non-existen-image.jpeg'
      />
    </Div>
    <Div pushed>
      <Avatar
        fallback='John Doe'
        url='/img/non-existen-image.jpeg'
      />
    </Div>
  </Row>
)
```
