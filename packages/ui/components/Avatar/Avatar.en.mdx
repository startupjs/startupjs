# Avatar

Avatars can be used to display image, icons, letters.

## Basic Example

```jsx
import { Avatar } from '@startupjs/ui'
```

```jsx example
return (
  <Avatar
    url='/img/avatar1.jpeg'
  />
)
```

## Avatar Status Example

```jsx
import { Avatar, Row, Div } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Div>
      <Avatar
        status='online'
        url='/img/avatar1.jpeg'
      />
    </Div>
    <Div pushed>
      <Avatar
        status='away'
        url='/img/avatar2.jpeg'
      />
    </Div>
  </Row>
)
```

## Avatar Fallback

When `url` prop is not provided or can't loaded intials of the `fallback` string are displayed.
In the example below `fallback` is 'John Doe' therefore 'JD' will be displayed.
Default `fallback` is `?`.

```jsx
import { Avatar, Row, Div } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Div>
      <Avatar
        url='/img/non-existen-image.jpeg'
      />
    </Div>
    <Div pushed>
      <Avatar
        fallback='John Doe'
        url='/img/non-existen-image.jpeg'
      />
    </Div>
  </Row>
)
```
