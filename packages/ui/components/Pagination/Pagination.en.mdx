# Pagination

The Pagination component enables the user to select a specific page from a range of pages.

## Example

```jsx
import { Pagination, Div } from '@startupjs/ui'
```

```jsx example
const PATH = '_session.ExamplePagination'
const $page = useModel(PATH)
useEffect(() => {
  $page.set(1)
}, [])
return (
  <Div style={{alignItems: 'center'}}>
    <Pagination
      value={$page.get()}
      total={10}
      onChange={(value)=>{$page.set(value)}}
    />
  </Div>
)
```

## Variant

The `variant` property specifies the type of component style, with two variants of type `filled` (default) and `floating`.

## Visible Count

The `visibleCount` property specifies the number of visible page buttons in the total range, with a default value of `3`.
