# TextInput

TextInput lets user enter or edit text.

```jsx
import { TextInput } from '@startupjs/ui'
```

## Example

```jsx example
const [value, setValue] = useState()
const [value1, setValue1] = useState()
return (
  <Div>
    <TextInput
      label='Example'
      placeholder='placeholder'
      value={value}
      onChangeText={setValue}
      onFocus={(e) => console.log('focused', e)}
      onBlur={(e) => console.log('blurred', e)}
    />
    <Br />
    <TextInput
      disabled
      label='Disabled input example'
      value='Disabled value'
    />
  </Div>
)
```

## Sizes

Size can be modified using the `size` prop. Default size is `m`.

```jsx example
const [valueL, setValueL] = useState()
const [valueM, setValueM] = useState()
const [valueS, setValueS] = useState()
return (
  <Div>
    <TextInput
      size='s'
      label='Size s example'
      value={valueS}
      onChangeText={setValueS}
    />
    <Br />
    <TextInput
      size='m'
      label='Size m (default) example'
      value={valueM}
      onChangeText={setValueM}
    />
    <Br />
    <TextInput
      size='l'
      label='Size l example'
      value={valueL}
      onChangeText={setValueL}
    />
  </Div>
)
```

## Icons

To display icon pass the `icon` property. Position of icon can be changed by passing `iconPosition` to component. To handle clicks on the icon, use the property `onIconPress`. By passing `iconStyle` to component you can change icon styles, for example color.

In `.styl` file
```stylus
.icon
  color: $UI.colors.primary
```

```jsx example
const [valueLeft, setLeftValue] = useState()
const [valueRight, setRightValue] = useState()
return (
  <Div>
    <TextInput
      icon={faTimesCircle}
      iconStyleName='icon'
      label='Example icon left'
      value={valueLeft}
      onChangeText={setLeftValue}
      onIconPress={() => setLeftValue()}
    />
    <Br />
    <TextInput
      icon={faTimesCircle}
      iconPosition='right'
      label='Example icon right'
      value={valueRight}
      onChangeText={setRightValue}
      onIconPress={() => setRightValue()}
    />
  </Div>
)
```

## Multiline mode (textarea)

The `multiline` property activates multiline mode.

Properties that only work in multiline mode:
- `numberOfLines` sets the number of visible lines
- `resize` the textinput height will grow with content

```jsx example
const [value, setValue] = useState()
return (
  <TextInput
    multiline
    resize
    numberOfLines={4}
    label='Multiline example'
    placeholder='Multiline placeholder'
    value={value}
    onChangeText={setValue}
  />
)
```

## Layout

There are two layouts. The default behavior is `rows` if `label` property provided and `pure` if not. To change the layout pass the property `layout`.

- `pure` displays text input without `label`
- `rows` displays text input with `label` and if `label` not set it using placeholder instead

```jsx example
const [pureValue, setPureValue] = useState()
const [rowValueWithLabel, setRowValueWithLabelValue] = useState()
const [rowValueWithoutLabel, setRowValueWithoutLabel] = useState()
return (
  <Div>
    <TextInput
      layout='pure'
      label='Pure layout example'
      placeholder='Pure layout placeholder'
      value={pureValue}
      onChangeText={setPureValue}
    />
    <Br />
    <TextInput
      label='Row layout bacause label provided'
      placeholder='Row layout placeholder with label'
      value={rowValueWithLabel}
      onChangeText={setRowValueWithLabelValue}
    />
    <Br />
    <TextInput
      layout='rows'
      placeholder='Row layout placeholder without label'
      value={rowValueWithoutLabel}
      onChangeText={setRowValueWithoutLabel}
    />
  </Div>
)
```
