# TextInput (текстовое поле)

Текстовое поле позволяет пользователю вводить или редактировать текст.

```jsx
import { TextInput } from '@startupjs/ui'
```

## Простой пример

```jsx example
const [value, setValue] = useState()
return (
  <TextInput
    placeholder='TextInput'
    value={value}
    onChangeText={setValue}
  />
)
```

## Label

```jsx example
const [value, setValue] = useState()
return (
  <TextInput
    label='Label'
    placeholder='TextInput'
    value={value}
    onChangeText={setValue}
  />
)
```

## Disabled

```jsx example
return (
  <TextInput
    disabled
    value='disabled'
  />
)
```

## Readonly

```jsx example
return (
  <TextInput
    readonly
    value='readonly'
  />
)
```

## Размеры

Размер может быть изменен с помощью свойства `size`. Размер по умолчанию - `m`.

```jsx example
const [valueL, setValueL] = useState()
const [valueM, setValueM] = useState()
const [valueS, setValueS] = useState()
return (
  <Div>
    <TextInput
      size='s'
      label='Size s example'
      value={valueS}
      onChangeText={setValueS}
    />
    <Br />
    <TextInput
      size='m'
      label='Size m (default) example'
      value={valueM}
      onChangeText={setValueM}
    />
    <Br />
    <TextInput
      size='l'
      label='Size l example'
      value={valueL}
      onChangeText={setValueL}
    />
  </Div>
)
```

## Иконки

Иконка может быть добавлена используя `icon` и `secondaryIcon` свойства. Положение иконки можно изменить, передав в компонент `iconPosition` (`left` по умолчанию). `secondaryIcon` использует противоположное положение `iconPosition`. Для обработки нажатий на иконке используйте свойство `onIconPress` и `onSecondaryIconPress`. Чтобы изменить цвет иконки используйте `iconStyleName` и `secondaryIconStyleName` свойства.

В `.styl` файле
```stylus
.icon
  color: $UI.colors.primary
```

```jsx example
const [valueLeft, setLeftValue] = useState()
const [valueRight, setRightValue] = useState()
return (
  <Div>
    <TextInput
      icon={faSearch}
      secondaryIcon={faTimesCircle}
      secondaryIconStyleName='icon'
      label='Example icon left'
      value={valueLeft}
      onChangeText={setLeftValue}
      onSecondaryIconPress={() => setLeftValue()}
    />
    <Br />
    <TextInput
      icon={faTimesCircle}
      iconPosition='right'
      label='Example icon right'
      value={valueRight}
      onChangeText={setRightValue}
      onIconPress={() => setRightValue()}
    />
  </Div>
)
```

## Многострочный режим (textarea)

Свойство `multiline` активирует многострочный режим.

Свойства которые работают только в многострочном режиме:
- `numberOfLines` задает количество видимых линий
- `resize` высота текстового поля будет расти вместе с содержимым

```jsx example
const [value, setValue] = useState()
return (
  <TextInput
    multiline
    resize
    numberOfLines={4}
    label='Multiline example'
    placeholder='Multiline placeholder'
    value={value}
    onChangeText={setValue}
  />
)
```

## Компоновка

Сущесвтует две компоновки. Поведение по умолчанию - `rows` если `label` свойтсво задано и `pure` если нет. Чтобы изменить компоновку передайте свойство `layout`.

- `pure` отображает текстовое поле без `label`
- `rows` отображает текстовое поле с `label` и если `label` не задан, то вместо нее используется `placeholder`

```jsx example
const [pureValue, setPureValue] = useState()
const [rowValueWithLabel, setRowValueWithLabelValue] = useState()
const [rowValueWithoutLabel, setRowValueWithoutLabel] = useState()
return (
  <Div>
    <TextInput
      layout='pure'
      label='Pure layout example'
      placeholder='Pure layout placeholder'
      value={pureValue}
      onChangeText={setPureValue}
    />
    <Br />
    <TextInput
      label='Row layout bacause label provided'
      placeholder='Row layout placeholder with label'
      value={rowValueWithLabel}
      onChangeText={setRowValueWithLabelValue}
    />
    <Br />
    <TextInput
      layout='rows'
      placeholder='Row layout placeholder without label'
      value={rowValueWithoutLabel}
      onChangeText={setRowValueWithoutLabel}
    />
  </Div>
)
```
