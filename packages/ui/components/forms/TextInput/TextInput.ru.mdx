# TextInput (поле ввода текста)

Поле ввода текста позволяет пользователю вводить или редактировать текст.

## Пример

```jsx
import { TextInput, Div, Br } from '@startupjs/ui'
```

```jsx example
const [value, setValue] = useState()
const [value1, setValue1] = useState()
return (
  <Div>
    <TextInput
      label='Example'
      placeholder='placeholder'
      value={value}
      onChangeText={setValue}
      onFocus={(e) => console.log('focused', e)}
      onBlur={(e) => console.log('blurred', e)}
    />
    <Br />
    <TextInput
      disabled
      label='Disabled input example'
      value='Disabled value'
    />
  </Div>
)
```

## Размеры

Размер может быть изменен с помощью свойства `size`. Размер по умолчанию - `m`.

## Sizes

Size can be modified using the `size` prop. Default size is `m`.

```jsx
import { TextInput, Div, Br } from '@startupjs/ui'
```

```jsx example
const [valueL, setValueL] = useState()
const [valueM, setValueM] = useState()
const [valueS, setValueS] = useState()
return (
  <Div>
    <TextInput
      size='s'
      label='Size s example'
      value={valueS}
      onChangeText={setValueS}
    />
    <Br />
    <TextInput
      size='m'
      label='Size m (default) example'
      value={valueM}
      onChangeText={setValueM}
    />
    <Br />
    <TextInput
      size='l'
      label='Size l example'
      value={valueL}
      onChangeText={setValueL}
    />
  </Div>
)
```

## Иконки

Для отображения иконки передайте свойство `icon`. Положение иконки может быть изменено путем передачи `iconPosition` в компонент. Для обработки нажатий на иконке используйте свойство `onIconPress`.

```jsx
import { TextInput, Div, Br } from '@startupjs/ui'
import { faTimesCircle } from '@fortawesome/free-solid-svg-icons'
```

```jsx example
const [valueLeft, setLeftValue] = useState()
const [valueRight, setRightValue] = useState()
return (
  <Div>
    <TextInput
      icon={faTimesCircle}
      label='Example icon left'
      value={valueLeft}
      onChangeText={setLeftValue}
      onIconPress={() => setLeftValue()}
    />
    <Br />
    <TextInput
      icon={faTimesCircle}
      iconPosition='right'
      label='Example icon right'
      value={valueRight}
      onChangeText={setRightValue}
      onIconPress={() => setRightValue()}
    />
  </Div>
)
```

## Многострочный режим (textarea)

Свойство `multiline` активирует многострочный режим.

Свойства которые работают только в многострочном режиме:
- `numberOfLines` задает количество видимых линий
- `resize` высота поля ввода текста будет расти вместе с содержимым


```jsx
import { TextInput } from '@startupjs/ui'
```

```jsx example
const [value, setValue] = useState()
return (
  <TextInput
    multiline
    resize
    numberOfLines={4}
    label='Multiline example'
    placeholder='Multiline placeholder'
    value={value}
    onChangeText={setValue}
  />
)
```

## Компоновка

Сущесвтует две компоновки. Поведение по умолчанию - `rows` если `label` свойтсво задано и `pure` если нет. Чтобы изменить компоновку передайте свойство `layout`.

- `pure` отображает поле ввода текста без `label`
- `rows` отображает поле ввода текста с `label` и если `label` не задан, то вместо нее используется `placeholder`

```jsx
import { TextInput, Div, Br } from '@startupjs/ui'
```

```jsx example
const [pureValue, setPureValue] = useState()
const [rowValueWithLabel, setRowValueWithLabelValue] = useState()
const [rowValueWithoutLabel, setRowValueWithoutLabel] = useState()
return (
  <Div>
    <TextInput
      layout='pure'
      label='Pure layout example'
      placeholder='Pure layout placeholder'
      value={pureValue}
      onChangeText={setPureValue}
    />
    <Br />
    <TextInput
      label='Row layout bacause label provided'
      placeholder='Row layout placeholder with label'
      value={rowValueWithLabel}
      onChangeText={setRowValueWithLabelValue}
    />
    <Br />
    <TextInput
      layout='rows'
      placeholder='Row layout placeholder without label'
      value={rowValueWithoutLabel}
      onChangeText={setRowValueWithoutLabel}
    />
  </Div>
)
```
