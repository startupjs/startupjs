# ColorPicker (выбор цвета)

ColorPicker позволяет пользователю выбрать цвет из цветовой палитры (работает с цветами в шестнадцатеричном формате).

## Импорт

```jsx
import { ColorPicker } from '@startupjs/ui'
```

## Простой пример

```jsx example
const [color, setColor] = useState()

return (
  <ColorPicker
    value={color}
    onChangeColor={setColor}
  />
)
```
