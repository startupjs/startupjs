# ColorPicker (выбор цвета)

ColorPicker позволяет пользователю выбрать цвет из цветовой палитры (работает с цветами в шестнадцатеричном формате).

## Импорт

```jsx
import { ColorPicker } from '@startupjs/ui'
```

## Простой пример

```jsx example
const [color, setColor] = useState()

return (
  <ColorPicker
    value={color}
    onChangeColor={setColor}
  />
)
```

## Свойства

- ```value``` (String) - исходный цвет в шестнадцатеричном формате (по умолчанию ```#fff```);
- ```disabled``` (Boolean) - отключен ли пикер (по умолчанию ```false```);
- ```size``` (```s```|```m```|```l```) - размер кнопки пикера (по умолчанию ```m```);
- ```onChangeColor``` (Function) - обработчик выбора цвета, возвращает цвет в шестнадцатеричном формате.
