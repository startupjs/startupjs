# ColorPicker

ColorPicker allows user to select a color from the color palette (works with colors in hexadecimal format).

## Import

```jsx
import { ColorPicker } from '@startupjs/ui'
```

## Simple example

```jsx example
const [color, setColor] = useState()

return (
  <ColorPicker
    value={color}
    onChangeColor={setColor}
  />
)
```
