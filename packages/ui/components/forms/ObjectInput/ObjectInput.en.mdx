# ObjectInput

ObjectInput lets build dynamic forms using its declarative api.

```jsx
import { ObjectInput } from from '@startupjs/ui'
```

## Example

ObjectInput accepts an object in the `properties` property with a required `input` key to specify what type of input to display. Possible types are ([—Åheckbox](/docs/components/Checkbox), [text](/docs/components/TextInput) & [object](/docs/components/ObjectInput)). The `properties` object also inherit any of the props passed to the these components.

```jsx example
const [value, $value] = useValue()
return (
  <ObjectInput
    type='text'
    value={value}
    $value={$value}
    properties={{
      firstName: {
        input: 'text',
        label: 'First name'
      },
      lastName: {
        input: 'text',
        label: 'Last name'
      },
      termsAndCondition: {
        input: 'checkbox',
        label: 'I agree to T&C'
      }
    }}
  />
)
```

## Inputs order

ObjectInput accepts an array in the `order` property to specify in what order should the inputs be displayed.

```jsx example
const [value, $value] = useValue()
return (
  <ObjectInput
    type='text'
    value={value}
    $value={$value}
    order={['lastName', 'termsAndCondition', 'firstName']}
    properties={{
      firstName: {
        input: 'text',
        label: 'First name'
      },
      lastName: {
        input: 'text',
        label: 'Last name'
      },
      termsAndCondition: {
        input: 'checkbox',
        label: 'I agree to T&C'
      }
    }}
  />
)
```

## Advanced usage

ObjectInput supports `dependsOn` and `dependsValue` properties for each object from `properties` array to dynamically display inputs.

- `dependsOn` is used to specify the object key which the current input depends on
- `dependsValue` is used to specify at what value of the input with key `dependsOn` should the current input be shown

In the `example-1` below the dependent textbox will only be shown when the default textbox is non-empty. Similarly, `example-2` only shows dependent textbox when the checkbox is checked.

Caution: When the dependsOn field changes and dependent field is no longer visible its previous value is preserved.

```jsx example
const [value, $value] = useValue({})
const [value2, $value2] = useValue()
return (
  <Content>
    <Span>Example-1</Span>
    <Br lines={0.5} />
    <ObjectInput
      type='text'
      value={value}
      $value={$value}
      properties={{
        textInput: {
          input: 'text',
          label: 'Text input',
          placeholder: 'Type anything'
        },
        dependent: {
          dependsOn: 'textInput',
          dependsValue: value.textInput,
          input: 'text',
          label: 'Dependent on text input'
        }
      }}
    />
    <Br />
    <Span>Example-2</Span>
    <Br lines={0.5}/>
    <ObjectInput
      type='text'
      value={value2}
      $value={$value2}
      properties={{
        checkbox: {
          input: 'checkbox',
          label: 'Click to display next input'
        },
        dependent: {
          dependsOn: 'checkbox',
          dependsValue: true,
          input: 'text',
          label: 'Dependent on checkbox'
        }
      }}
    />
  </Content>
)
```
