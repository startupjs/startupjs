# Input (инпут)

Инпут предоставляет апи-обвертку над [Checkbox](/docs/forms/Checkbox), [TextInput](/docs/forms/TextInput), [Select](/docs/forms/Select) и [ObjectInput](/docs/forms/ObjectInput) компонентами, добавляя двустороннию привязку данных. Вы можете использовать любой из вышеперечисленных компонентов, указав свойство `type`.

```jsx
import { Input } from from '@startupjs/ui'
```

## Пример

```jsx example
const [, $textValue] = useValue()
const [, $checkboxValue] = useValue()
const [, $objectValue] = useValue()
return (
  <Div>
    <Input
      type='text'
      label='Text input'
      $value={$textValue}
    />
    <Br />
    <Input
      type='checkbox'
      label='Checkbox component'
      $value={$checkboxValue}
    />
    <Br />
    <Input
      type='object'
      $value={$objectValue}
      properties={{
        checkbox: {
          input: 'checkbox',
          label: 'Click to display next input'
        },
        dependent: {
          dependsOn: 'checkbox',
          dependsValue: true,
          input: 'text',
          label: 'Dependent on checkbox',
          style: { marginTop: u(1) }
        }
      }}
    />
  </Div>
)
```

## Сложный пример

При использовании `object` инпута, важно объявить `schema` как константу либо вне вашего компонента, либо закэшировать ее через `useMemo()`.
Если этого не сделать, то `Input` будет перерендеривать все дерево формы целиком при редактировании лишь одного поля.

```jsx example
const schema = useMemo(() => ({
  name: {
    type: 'string',
    label: 'Name'
  },
  number: {
    type: 'number',
    label: 'Level'
  },
  dangerous: {
    type: 'boolean',
    label: 'Is Dangerous'
  },
  type: {
    type: 'select',
    label: 'Type',
    options: ['Fire', 'Water', 'Earth', 'Electric']
  },
  address: {
    type: 'object',
    label: 'Address',
    properties: {
      galaxy: {
        type: 'string',
        label: 'Galaxy'
      },
      planet: {
        type: 'number',
        label: 'Planet Number'
      },
      location: {
        type: 'string',
        label: 'Location'
      }
    }
  },
  abilities: {
    type: 'array',
    label: 'Abilities',
    items: { type: 'string' }
  },
  url: {
    type: 'string',
    label: 'Image URL'
  }
}), [])

const [value, $value] = useValue()

return (
  <React.Fragment>
    <Input
      type='object'
      $value={$value}
      properties={schema}
    />
    <Br />
    <Span style={{ fontFamily: 'monospace' }}>
      VALUE: {JSON.stringify(value, null, 2)}
    </Span>
  </React.Fragment>
)
```

## Отображение ошибок

Свойство `error` предоставляет текст ошибки для инпута.

```jsx example
const [value, $value] = useValue()

return (
  <Input
    $value={$value}
    error={value ? '' : 'Need to provide value'}
  />
)
```
