# Input (инпут)

Инпут предоставляет апи-обвертку над [Checkbox](/docs/forms/Checkbox), [TextInput](/docs/forms/TextInput), [Select](/docs/forms/Select) и [ObjectInput](/docs/forms/ObjectInput) компонентами, добавляя двустороннию привязку данных. Вы можете использовать любой из вышеперечисленных компонентов, указав свойство `type`.

```jsx
import { Input } from from '@startupjs/ui'
```

## Пример

```jsx example
const [, $textValue] = useValue()
const [, $checkboxValue] = useValue()
const [, $objectValue] = useValue()
return (
  <Div>
    <Input
      type='text'
      label='Text input'
      $value={$textValue}
    />
    <Br />
    <Input
      type='checkbox'
      label='Checkbox component'
      $value={$checkboxValue}
    />
    <Br />
    <Input
      type='object'
      $value={$objectValue}
      properties={{
        checkbox: {
          input: 'checkbox',
          label: 'Click to display next input'
        },
        dependent: {
          dependsOn: 'checkbox',
          dependsValue: true,
          input: 'text',
          label: 'Dependent on checkbox',
          style: { marginTop: u(1) }
        }
      }}
    />
  </Div>
)
```

## Сложный пример

```jsx example
const schema = {
  name: {
    type: 'string',
    label: 'Name'
  },
  number: {
    type: 'number',
    label: 'Level'
  },
  type: {
    type: 'select',
    label: 'Type',
    options: ['Fire', 'Water', 'Earth', 'Electric']
  },
  abilities: {
    type: 'object',
    label: 'Abilities',
    properties: {
      ability0: {
        type: 'string',
        label: 'Ability 1'
      },
      ability1: {
        type: 'string',
        label: 'Ability 2'
      },
      ability2: {
        type: 'string',
        label: 'Ability 3'
      }
    }
  },
  url: {
    type: 'string',
    label: 'Image URL'
  },
  dangerous: {
    type: 'boolean',
    label: 'Is Dangerous'
  }
}

const [value, $value] = useValue()

return (
  <React.Fragment>
    <Input
      type='object'
      $value={$value}
      properties={schema}
    />
    <Br />
    <Span style={{ fontFamily: 'monospace' }}>
      VALUE: {JSON.stringify(value, null, 2)}
    </Span>
  </React.Fragment>
)
```
