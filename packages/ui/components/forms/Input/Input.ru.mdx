# Input (инпут)

Инпут предоставляет апи-обвертку над [Checkbox](/docs/ru/forms/Checkbox), [TextInput](/docs/ru/forms/TextInput), [Select](/docs/ru/forms/Select) и [ObjectInput](/docs/ru/forms/ObjectInput) компонентами, добавляя двустороннию привязку данных. Вы можете использовать любой из вышеперечисленных компонентов, указав свойство `type`.

```jsx
import { Input } from from '@startupjs/ui'
```

## Пример

```jsx example
const [, $textValue] = useValue()
const [, $checkboxValue] = useValue()
const [, $objectValue] = useValue()
return (
  <Div>
    <Input
      type='text'
      label='Text input'
      $value={$textValue}
    />
    <Br />
    <Input
      type='checkbox'
      label='Checkbox component'
      $value={$checkboxValue}
    />
    <Br />
    <Input
      type='object'
      $value={$objectValue}
      properties={{
        checkbox: {
          input: 'checkbox',
          label: 'Click to display next input'
        },
        dependent: {
          dependsOn: 'checkbox',
          dependsValue: true,
          input: 'text',
          label: 'Dependent on checkbox',
          style: { marginTop: u(1) }
        }
      }}
    />
  </Div>
)
```
