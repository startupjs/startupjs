# Input

Input provides a wrapper api around [Checkbox](/docs/forms/Checkbox), [TextInput](/docs/forms/TextInput), [Select](/docs/forms/Select) & [ObjectInput](/docs/forms/ObjectInput) components by adding two-way data bindings. You can use any of the above components by specifying the `type` prop.

```jsx
import { Input } from from '@startupjs/ui'
```

## Example

```jsx example
const [, $textValue] = useValue()
const [, $checkboxValue] = useValue()
const [, $objectValue] = useValue()
return (
  <Div>
    <Input
      type='text'
      label='Text input'
      $value={$textValue}
    />
    <Br />
    <Input
      type='checkbox'
      label='Checkbox component'
      $value={$checkboxValue}
    />
    <Br />
    <Input
      type='object'
      $value={$objectValue}
      properties={{
        checkbox: {
          input: 'checkbox',
          label: 'Click to display next input'
        },
        dependent: {
          dependsOn: 'checkbox',
          dependsValue: true,
          input: 'text',
          label: 'Dependent on checkbox',
          style: { marginTop: u(1) }
        }
      }}
    />
  </Div>
)
```

## Complex example

```jsx example
const schema = {
  name: {
    type: 'string',
    label: 'Name'
  },
  number: {
    type: 'string',
    label: 'Number'
  },
  abilities: {
    type: 'object',
    label: 'Abilities',
    properties: {
      ability0: {
        type: 'string',
        label: 'Ability 1'
      },
      ability1: {
        type: 'string',
        label: 'Ability 2'
      },
      ability2: {
        type: 'string',
        label: 'Ability 3'
      }
    }
  },
  url: {
    type: 'string',
    label: 'Image URL'
  },
  dangerous: {
    type: 'boolean',
    label: 'Is Dangerous'
  }
}

const [, $value] = useValue()

return (
  <Input
    type='object'
    $value={$value}
    properties={schema}
  />
)
```
