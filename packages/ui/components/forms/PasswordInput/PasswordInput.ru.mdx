# PasswordInput

PasswordInput предоставляет текстовое поле для ввода пароля c кнопкой скрытия вводимого текста.
PasswordInput наследует свойства компонента [TextInput](/docs/components/TextInput).

```jsx
import { PasswordInput } from from '@startupjs/ui'
```

## Простой пример

```jsx example
const [password, setPassword] = useState()

return (
  <PasswordInput
    value={password}
    onChangeText={setPassword}
  />
)
```

## Label and description

```jsx example
const [password, setPassword] = useState()

return (
  <PasswordInput
    label='Password'
    description="Make sure it's at least 15 characters OR at least 8 characters including a number and a lowercase letter"
    value={password}
    onChangeText={setPassword}
  />
)
```

## Disabled

```jsx example
return (
  <PasswordInput
    disabled
    value='Disabled'
  />
)
```
