# Select

Multiselect lets user pick multiple options.

```jsx
import { Multiselect } from '@startupjs/ui'
```

## Initialization

Before use you need to configure [Portal](/docs/components/Portal)

## Example

- `options` - array of objects ```{ label, value }``` or primitives
- `TagComponent` - prop for custom tag render
- `tagLimit` - max amount of displaying tags
- `onChange` ```function(value: Array)``` - the callback triggered when the value is changed. Returns an array with selected values

```jsx example
const OPTIONS = [
  { label: 'New York', value: 'ny' },
  { label: 'Los Angeles', value: 'la' },
  { label: 'Tokyo', value: 'tk' },
  { label: 'Moscow', value: 'msc' },
  { label: 'Berlin', value: 'bl' },
  { label: 'Milan', value: 'mi' },
  { label: 'Sydney', value: 'sy' },
  { label: 'Porto', value: 'po' }
]

const [cities, setCities] = useState(['la', 'tk', 'msc', 'po', 'ny'])

return (
  <Div>
    <Multiselect
      label='Example'
      placeholder='Select cities'
      value={cities}
      onChange={setCities}
      options={OPTIONS}
    />
    <Br />
    <Multiselect
      label='Example with tags limiter'
      placeholder='Select cities'
      tagLimit={3}
      value={cities}
      onChange={setCities}
      options={OPTIONS}
    />
    <Br />
    <Multiselect
      label='Example Multiselect with custom tag'
      value={cities}
      TagComponent={({ record }) => <Span>âœ” {record.label} &nbsp;&nbsp;&nbsp;&nbsp;</Span>}
      onChange={setCities}
      options={OPTIONS}
    />
    <Br />
    <Multiselect
      label='Example of disabled Multiselect'
      placeholder='Select cities'
      value={cities}
      options={OPTIONS}
      disabled
    />
    <Br />
    <Multiselect
      label='Example of readonly Multiselect'
      placeholder='Select cities'
      value={cities}
      options={OPTIONS}
      readonly
    />
  </Div>
)
```
