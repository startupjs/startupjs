# Select

Multiselect lets user pick multiple options.

```jsx
import { Multiselect } from '@startupjs/ui'
```

- `options` - array of objects ```{ label, value }``` or primitives
- `onChange` - the callback triggered when the value is changed. Returns an array with selected values | ```function(value: Array)```

```jsx example
const [cities, setCities] = useState(['la', 'tk'])
// options also can be an array of primitives ['apple', 'banana', 'cherry']

return (
  <Div>
    <Multiselect
      label='Example'
      placeholder='Select cities'
      value={cities}
      onChange={setCities}
      options={[
        { label: 'New York', value: 'ny' },
        { label: 'Los Angeles', value: 'la' },
        { label: 'Tokyo', value: 'tk' }
      ]}
    />
    <Br />
    <Multiselect
      label='Example readonly Multiselect'
      readonly
      value={cities}
      options={[
        { label: 'New York', value: 'ny' },
        { label: 'Los Angeles', value: 'la' },
        { label: 'Tokyo', value: 'tk' }
      ]}
    />
  </Div>
)
```

## Custom tag

You can pass custom component to TagComponent property that displays instead of default tag.

```jsx example
const [cities, setCities] = useState(['la', 'tk'])
return (
  <Div>
    <Multiselect
      label='Example Multiselect with custom tag'
      value={cities}
      TagComponent={({ record }) => <Span>âœ” {record.label} </Span>}
      onChange={setCities}
      options={[
        { label: 'New York', value: 'ny' },
        { label: 'Los Angeles', value: 'la' },
        { label: 'Tokyo', value: 'tk' }
      ]}
    />  
  </Div>  
)
```
