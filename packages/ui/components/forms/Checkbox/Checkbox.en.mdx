# Checkbox

Checkboxes allow the user to select one or more items from a set.

```jsx
import { Checkbox } from from '@startupjs/ui'
```

## Example

```jsx example
const [checked, setChecked] = useState(false)
return (
  <Div>
    <Checkbox
      label='Example'
      value={checked}
      onChange={setChecked}
    />
    <Br />
    <Checkbox
      label='Disabled example'
      value={true}
      disabled
    />
    <Br />
    <Checkbox
      label='Readonly example'
      value={true}
      readOnly
    />
  </Div>
)
```

## Switch

To display component as the switch pass `variant='switch'` to it.

```jsx example
const [checked, setChecked] = useState(false)
return (
  <Div>
    <Checkbox
      variant='switch'
      label='Switch example'
      value={checked}
      onChange={setChecked}
    />
    <Br />
    <Checkbox
      variant='switch'
      label='Switch disbabled example'
      value={true}
      disabled
    />
    <Br />
    <Checkbox
      variant='switch'
      label='Switch readonly example'
      value={true}
      readOnly
    />
  </Div>
)
```
