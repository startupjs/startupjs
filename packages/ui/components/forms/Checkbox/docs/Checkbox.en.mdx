# Checkbox

Checkboxes allow the user to select one or more items from a set.

```jsx
import { Checkbox } from from '@startupjs/ui'
```

## Simple example

```jsx example
const [checked, setChecked] = useState(false)

return (
  <Div>
    <Checkbox
      value={checked}
      onChange={setChecked}
    />
    <Br />
    <Checkbox
      variant='switch'
      value={checked}
      onChange={setChecked}
    />
  </Div>
)
```

## Variants

There are two display variants of component: `checkbox` and `switch`. **Default**: `checkbox`.

```jsx example
const [checked, setChecked] = useState(false)

return (
  <Checkbox
    variant='switch'
    value={checked}
    onChange={setChecked}
  />
)
```


## Label and description

```jsx example
const [checked, setChecked] = useState(false)

return (
  <Checkbox
    label='Mark absent when away'
    description='Mark student absent if he goes offline for more then 2 minutes'
    value={checked}
    onChange={setChecked}
  />
)
```

## Disabled

```jsx example
return (
  <Checkbox disabled />
)
```

## Readonly

```jsx example
return (
  <Checkbox readonly />
)
```

## Checkbox with custom icon

The `icon` property allows to change the default icon.

```jsx example
const [checked, setChecked] = useState(false)

return (
  <Checkbox
    value={checked}
    icon={faMinus}
    onChange={setChecked}
  />
)
```
