# RangeInput

The component allow users to make selections from a range of values as a single value or a range.

```jsx
import { RangeInput } from '@startupjs/ui'
```

## Simple example
By default range input has one slider. `value` prop must be a number in this case. Otherwise you will receive a waring if `value` prop is an array. But the component is smart enough and it will still work.

**Important**: `onChange` callback receives first argument with type number

If value prop hasn't been provided then a value of `min` prop will be used as a value for `value` prop. **`onChange` callback prop will be called to initialize your model with that value**

```jsx example
const [value, setValue] = useState(40)

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    onChange={(val) => setValue(val)}
  />
)
```

## Two sliders
To enable two sliders specify `range` prop with `true` value.
`value` prop must be an array with two numbers.

**Important**: In this case `onChange` callback receives first argument with array type

If value hasn't been provided then a values of `min` and `max` props will be used as a value for `value` prop.
**`onChange` callback prop will be called to initialize your model with that value**
Even if you pass a value with one element the component will use a value of `max` prop to fill absent second element but you will receive a warning that value prop must has two elements in the array.

```jsx example
const [value, setValue] = useState([20, 60])

return (
  <RangeInput
    min={0}
    max={100}
    range
    value={value}
    onChange={(val) => setValue(val)}
  />
)
```

## Disabling label(s)

Specify `showLabel` property with `false` value to hide label(s)

```jsx example
const [value, setValue] = useState(40)

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    showLabel={false}
    onChange={(val) => setValue(val)}
  />
)
```

## You can change a step for a value(s)

Use `step` property to specify a different from `1` value

```jsx example
const [value, setValue] = useState(30)

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    step={10}
    onChange={(val) => setValue(val)}
  />
)
```

## Show labels for steps
Use `showSteps` property with `true` value

```jsx example
const [value, setValue] = useState(4)

return (
  <RangeInput
    value={value}
    min={0}
    max={5}
    showSteps
    onChange={(val) => setValue(val)}
  />
)
```

## Range with specific width

Use `width` property to specify custom width

```jsx example
const [value, setValue] = useState(30)

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    width={400}
    onChange={(val) => setValue(val)}
  />
)
```

## Custom style

You can change styles for container, step, track and marker.
It is `containerStyle`, `stepStyle`, `trackStyle` and `markerStyle` props respectively
