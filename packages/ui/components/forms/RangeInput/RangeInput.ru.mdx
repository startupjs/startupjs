# RangeInput

Компонент позволяет делать выбор из диапазона значений

```jsx
import { RangeInput } from '@startupjs/ui'
```

## Простой пример
Когда значение является числом, а не массивом, поле имеет один ползунок

**Важно**: В этом случае обратный вызов `onChange` получает первый аргумент как число

```jsx example
const [value, setValue] = useState(10)

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    onChange={(val) => setValue(val)} // val is number
  />
)
```

## Два ползунка
Когда значение представляет собой массив (массив с двумя числами), поле имеет два ползунка.

**Important**: In this case `onChange` callback receives first argument with array type
**Важно**: В этом случае обратный вызов `onChange` получает первый аргумент в виде массива значений

```jsx example
const [value, setValue] = useState([20, 60])

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    onChange={(val) => setValue(val)}
  />
)
```

## Выключение показа значения(ий) для ползунка(ов)

Укажите свойство `hideLabel` чтобы скрыть значение(ия)

```jsx example
const [value, setValue] = useState(40)
const [secondValue, secondSetValue] = useState([40, 60])

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    hideLabel
    onChange={(val) => setValue(val)}
  />
)
```

## Вы можете изменить шаг для ползунка(ов)

Используйте свойство `step`, чтобы указать значение, отличное от `1`

```jsx example
const [value, setValue] = useState(30)

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    step={10}
    onChange={(val) => setValue(val)}
  />
)
```

## Вы можете указать список доступных значений для поля

Используйте свойство `options`, чтобы указать их.

**Важно:** свойства `min` и `max` будут игнорироваться, если они были указаны

```jsx example
const [value, setValue] = useState(30)

return (
  <RangeInput
    value={value}
    options={[0, 20, 40, 70, 90, 100]}
    onChange={(val) => setValue(val)}
  />
)
```

## Поле с определенной шириной

Используйте свойство `width`, чтобы указать пользовательскую ширину

```jsx example
const [value, setValue] = useState(30)

return (
  <RangeInput
    value={value}
    min={0}
    max={100}
    width={400}
    onChange={(val) => setValue(val)}
  />
)
```

## Пользовательский стиль

Вы можете изменить стили для контейнера, шага, дорожки и ползунка.
Это свойства `containerStyle`, `stepStyle`, `trackStyle` и `markerStyle` соответственно.
