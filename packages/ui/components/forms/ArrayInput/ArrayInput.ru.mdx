# ArrayInput

ArrayInput позволяет создавать динамические формы, используя его декларативный api.

ArrayInput принимает объект в свойстве `items` с метаданными инпута и обязательным ключом `input`, чтобы указать, какой тип ввода отображать.

Возможные типы:

[сheckbox](/docs/forms/Checkbox)
[text](/docs/forms/TextInput)
[object](/docs/forms/ObjectInput)
[select](/docs/forms/Select)
[date](/docs/forms/DateTimePicker)
[datetime](/docs/forms/DateTimePicker)
[time](/docs/forms/DateTimePicker)


```jsx
import { ArrayInput } from '@startupjs/ui'
```

## Простой пример

```jsx example
const [, $value] = useValue(['Green', 'Blue'])
return (
  <ArrayInput
    $value={$value}
    label='Colors'
    items={{
      input: 'text'
    }}
  />
)
```
