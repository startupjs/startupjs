# ArrayInput

ArrayInput позволяет создавать динамические формы, используя его декларативный апи.
ArrayInput принимает объект в свойстве `items` с обязательным ключом` input`, чтобы указать, какой тип ввода отображать.

Возможные типы:

[сheckbox](/docs/forms/Checkbox)
[text](/docs/forms/TextInput)
[object](/docs/forms/ObjectInput)
[select](/docs/forms/Select)
[date](/docs/forms/DateTimePicker)
[datetime](/docs/forms/DateTimePicker)
[time](/docs/forms/DateTimePicker)

Объект `items` также наследует любые свойства, переданные этим компонентам. 

```jsx
import { ArrayInput } from '@startupjs/ui'
```

## Простой пример

```jsx example
const [value, $value] = useValue(['Green', 'Blue'])
return (
  <ArrayInput
    value={value}
    $value={$value}
    label='Colors'
    items={{
      input: 'text'
    }}
  />
)
```
