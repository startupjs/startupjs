# NumberInput

NumberInput allows user to enter or edit number.

```jsx
import { NumberInput } from '@startupjs/ui'
```

## Example

```jsx example
const [value, setValue] = useState()
const [value1, setValue1] = useState(10)
return (
  <Div>
    <NumberInput
      label='NumberInput example'
      placeholder='placeholder'
      value={value}
      onChangeNumber={setValue}
      onFocus={(e) => console.log('focused', e)}
      onBlur={(e) => console.log('blurred', e)}
    />
    <Br />
    <NumberInput
      label='NumberInput with start value example'
      value={value1}
      onChangeNumber={setValue1}
    />
    <Br />
    <NumberInput
      disabled
      label='Disabled NumberInput example'
      value={value1}
    />
    <Br />
    <NumberInput
      readonly
      label='Readonly NumberInput with label example'
      value={value1}
    />
  </Div>
)
```

## Step

The step specifies how many decimal places can be entered, set by the numbers 1, 0.1, 0.01, etc. in the `step` property (1 by default).

```jsx example
const [value, setValue] = useState()
return (
  <Div>
    <NumberInput
      label='Step example'
      value={value}
      step={0.001}
      onChangeNumber={setValue}
    />
  </Div>
)
```

## Minimum and maximum value

The minimum and maximum values are set by the `min` and `max` properties

```jsx example
const [value, setValue] = useState()
return (
  <Div>
    <NumberInput
      label='Min and max example'
      value={value}
      min={-30}
      max={20}
      onChangeNumber={setValue}
    />
  </Div>
)
```

## Buttons

Buttons allow user to change the number at the specified step. The position of the buttons can be changed by passing the `buttons` property with the value `'vertical'` or `'horizontal'` (`'vertical'` by default) to the component or hidden by passing the `buttons` property with the value `'none'` to the component.

```jsx example
const [valueVertical, setValueVertical] = useState()
const [valueHorizontal, setValueHorizontal] = useState()
const [valueNone, setValueNone] = useState()
return (
  <Div>
    <NumberInput
      label='Vertical buttons example'
      buttons='vertical'
      value={valueVertical}
      onChangeNumber={setValueVertical}
    />
    <Br />
    <NumberInput
      label='Horizontal buttons example'
      buttons='horizontal'
      value={valueHorizontal}
      onChangeNumber={setValueHorizontal}
    />
    <Br />
    <NumberInput
      label='Without buttons example'
      buttons='none'
      value={valueNone}
      onChangeNumber={setValueNone}
    />
  </Div>
)
```

## Sizes

Size can be modified using the `size` prop. Default size is `'m'`.

```jsx example
const [valueL, setValueL] = useState()
const [valueM, setValueM] = useState()
const [valueS, setValueS] = useState()
return (
  <Div>
    <NumberInput
      size='s'
      label='Size s example'
      value={valueS}
      onChangeNumber={setValueS}
    />
    <Br />
    <NumberInput
      size='m'
      label='Size m (default) example'
      value={valueM}
      onChangeNumber={setValueM}
    />
    <Br />
    <NumberInput
      size='l'
      label='Size l example'
      value={valueL}
      onChangeNumber={setValueL}
    />
  </Div>
)
```

## Layout

There are two layouts. The default behavior is `'rows'` if `label` property provided and `'pure'` if not. To change the layout pass the property `layout`.

- `'pure'` displays text input without `label`
- `'rows'` displays text input with `label` and if `label` not set it using `placeholder` instead

```jsx example
const [pureValue, setPureValue] = useState()
const [rowValueWithLabel, setRowValueWithLabelValue] = useState()
const [rowValueWithoutLabel, setRowValueWithoutLabel] = useState()
return (
  <Div>
    <NumberInput
      layout='pure'
      label='Pure layout example'
      placeholder='Pure layout placeholder'
      value={pureValue}
      onChangeNumber={setPureValue}
    />
    <Br />
    <NumberInput
      label='Row layout because label provided'
      placeholder='Row layout placeholder with label'
      value={rowValueWithLabel}
      onChangeNumber={setRowValueWithLabelValue}
    />
    <Br />
    <NumberInput
      layout='rows'
      placeholder='Row layout placeholder without label'
      value={rowValueWithoutLabel}
      onChangeNumber={setRowValueWithoutLabel}
    />
  </Div>
)
```

## Stylization

For stylization can apply the properties:

- `style` - to style the root component
- `inputStyle` - to style the input
- `buttonStyle` - for styling buttons

```jsx example
const [value, setValue] = useState()

return (
    <NumberInput
      label='Custom style example'
      style={{
        width: 250
      }}
      buttonStyle={{
        borderWidth: 0
      }}
      inputStyle={{
        borderTopWidth: 0,
        borderLeftWidth: 0,
        borderRightWidth: 0,
        borderRadius: 0
      }}
      placeholder='Enter number'
      value={value}
      onChangeNumber={setValue}
    />
)
```
