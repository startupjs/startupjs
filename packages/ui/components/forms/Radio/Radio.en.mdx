# Radio

Radio button allows the user to select only one item from a list.

## Basic Example


```jsx
import { Radio } from '@startupjs/ui'
```

```jsx example
const [selectedItem, setSelectedItem] = useState('')
return (
  <Radio
    value={selectedItem}
    onChange={(value) => setSelectedItem(value)}
    data={[
      {
        value: 'Item-1',
        label: 'Item-1 label'
      },
      {
        value: 'Item-2',
        label: 'Item-2 label'
      },
      {
        value: 'Item-3',
        label: 'Item-3 label'
      },
      {
        value: 'Item-4',
        label: 'Item-4 label'
      }
    ]}
  />
)
```

## Radio Customization Example

Radio can be customized by directly passing children. Each child must have a unique `value` prop.

```jsx
import { Radio } from '@startupjs/ui'
```

```jsx
const CustomRadio = ({ value, onPress, label, checked }) => {
  return <Checkbox
    variant='switch'
    onChange={onPress}
    value={checked}
    label={label}
  />
}
```


```jsx example
const [selectedItem, setSelectedItem] = useState('')
return (
  <Radio
    value={selectedItem}
    onChange={(value) => setSelectedItem(value)}
  >
    <CustomRadioExample label='Item-1' value='Item-1' />
    <CustomRadioExample label='Item-2' value='Item-2' />
    <CustomRadioExample label='Item-3' value='Item-3' />
    <CustomRadioExample label='Item-4' value='Item-4' />
  </Radio>
)
```
