# Radio (радиокнопка)

Радиокнопки позволяют пользователю выбрать один вариант из списка.

```jsx
import { Radio } from from '@startupjs/ui'
```

## Пример

```jsx example
const [checked, setChecked] = useState()
return (
  <Radio
    value={checked}
    onChange={(value) => setChecked(value)}
    disabled={checked === 'item2'}
    options={[
      {
        value: 'item1',
        label: 'Item-1 label'
      },
      {
        value: 'item2',
        label: 'Disable items'
      },
      {
        value: 'item3',
        label: 'Item-3 label',
      },
      {
        value: 'item4',
        label: 'Item-4 disabled item',
        disabled: true
      }
    ]}
  />
)
```

## Radio.Item

Компонент, используемый для добавления пользовательской разметки в `Radio` компонент.

```jsx example
const [selectedItem, setSelectedItem] = useState('')
return (
  <Radio
    value={selectedItem}
    onChange={(value) => setSelectedItem(value)}
  >
    <Radio.Item value='item1'>
      <Span>Item-1</Span>
    </Radio.Item>
    <Radio.Item value='item2'>
      <Span>Item-2</Span>
    </Radio.Item>
    <Radio.Item value='item3'>
      <Span>Item-3</Span>
    </Radio.Item>
  </Radio>
)
```
