# DateTimePicker

DateTimePicker allows to display and change the date and time (!works with timestamp).

## Import

```jsx
import { DateTimePicker } from from '@startupjs/ui'
```

## Examples

```jsx example
const [dateTime, setDateTime] = useState()
const [date, setDate] = useState()
const [time, setTime] = useState()

return (
  <Div>
    <DateTimePicker
      date={dateTime}
      format='yyyy MM DD HH:mm'
      label='Date && Time'
      placeholder='Chose date and time'
      onDateChange={setDateTime}
    />
    <Br />
    <DateTimePicker
      date={date}
      format='MM-DD-yyyy'
      label='Date'
      maxDate={1607983200000}
      minDate={1605391200000}
      mode='date'
      size='l'
      onDateChange={setDate}
    />
    <Br />
    <DateTimePicker
      date={time}
      format='HH:mm'
      label='Time'
      minuteInterval={15}
      mode='time'
      size='s'
      timeFormat='HH-mm'
      onDateChange={setTime}
    />
  </Div>
)
```

## Options

- `date` - initial date or time in timestamp,
- `onDateChange` - date and time selection handler, returns a timestamp,
- `disabled` - disabled picker,
- `format` - date formatting string,
- `label` - label,
- `maxDate` - maximum available date in timestamp,
- `minDate` - minimum available date in timestamp,
- `minuteInterval` - the interval at which minutes can be selected, possible values are: 1, 2, 3, 4, 5, 6, 10, 12, 15, 20, 30 (default 1, ignored in IOS 14),
- `mode` - defines the picker type: `datetime` - date and time, `date` - date, `time` - time (default `datetime`),
- `placeholder` - placeholder for input on the web and button in the native application if the date is not selected,
- `size` - the sizes of the input on the web and the button in the native application, possible values are: `l`, `m` Ð¸ `s` (default `m`)

### Web only

- `timeFormat` - the format in which the time in the picker is displayed
- `customInput` - custom component instead of the standard `Input`

```jsx example
const [dateTime, setDateTime] = useState(Date.now())

const ExampleCustomInput = forwardRef(({ value, onClick }, ref) => {
  return (
    <Button color='primary' variant='flat' onPress={onClick}>
      {value}
    </Button>
  )
})

return (
  <Div>
    <DateTimePicker
      customInput={<ExampleCustomInput />}
      date={dateTime}
      mode='date'
      format='MMMM / DD / yyyy'
      label='Input as Button'
      onDateChange={setDateTime}
    />
  </Div>
)
```

### IOS only

- `cancelButtonText` - the text of the button that closes the modal,
- `confirmButtonText` - text of the button confirming the date change

### Android only

- `is24Hour` - if `true` allows changing of the time picker to a 24 hour format,
