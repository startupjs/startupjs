# Portal

```jsx
import { Portal } from '@startupjs/ui'
```

## Usage
1 - You need to initialize **Portal.Provider** at the very top of the application

For example in **Root**:
```jsx
return (
  <Portal.Provider>
    <App apps={{ main }} />
  </Portal.Provider>
)
```

Or **Layout**:
```jsx
function Layout ({ children }) {
  return (
    <Portal.Provider>
      {children}
    </Portal.Provider>
  )
}
```

2 - Throw something in the top of the application need use **Portal**. All those `children` that he receives are immediately displayed on top of everything

```jsx example
const [isShow, setIsShow] = useState(false)

const contentStyle = { position: 'absolute', height: 160, width: 200, backgroundColor: '#555555', right: 0, justifyContent: 'center', alignItems: 'center', borderRadius: 8 }

const textStyle = { color: '#fff', marginBottom: 16, fontSize: 24 }

return (
  <Div>
    <Button
      style={{ width: 120 }}
      onPress={()=> setIsShow(true)
    }>Open</Button>
    <Portal>
      {
        isShow &&
        <Div style={contentStyle}>
          <Span style={textStyle}>Content</Span>
          <Button
            variant='flat'
            onPress={()=> setIsShow(false)}
          >Close</Button>
        </Div>
      }
    </Portal>
  </Div>
)
```
