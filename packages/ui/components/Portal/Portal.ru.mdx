# Portal

```jsx
import { Portal } from '@startupjs/ui'
```

## Использование
1 - Нужно инициализоровать **Portal.Provider** в самом вверху приложения

Например в **Root**:
```jsx
return (
  <Portal.Provider>
    <App apps={{ main }} />
  </Portal.Provider>
)
```

Или **Layout**:
```jsx
function Layout ({ children }) {
  return (
    <Portal.Provider>
      {children}
    </Portal.Provider>
  )
}
```

2 - Чтобы прокинуть что-то в на самый вверх приложения, используется **Portal**. Все те `children` которые он получить, сразу же отображаются поверх всего

```jsx example
const [isShow, setIsShow] = useState(false)

const contentStyle = { position: 'absolute', height: 160, width: 200, backgroundColor: '#555555', right: 0, justifyContent: 'center', alignItems: 'center', borderRadius: 8 }

const textStyle = { color: '#fff', marginBottom: 16, fontSize: 24 }

return (
  <Div>
    <Button
      style={{ width: 120 }}
      onPress={()=> setIsShow(true)
    }>Открыть</Button>
    <Portal>
      {
        isShow &&
        <Div style={contentStyle}>
          <Span style={textStyle}>Контент</Span>
          <Button
            variant='flat'
            onPress={()=> setIsShow(false)}
          >Закрыть</Button>
        </Div>
      }
    </Portal>
  </Div>
)
```
