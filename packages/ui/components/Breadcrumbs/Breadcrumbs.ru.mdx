# Breadcrumbs (хлебные крошки)

Хлебные крошки указывают местоположение текущей страницы в навигационной иерархии и позволяют пользователям перемещаться по ней.

## Пример

Каждый объект маршрута принимает любые свойства компонента [`Link`](/docs/en/docs/Link) кроме тех что упомянуты ниже.

```jsx
import { Breadcrumbs } from '@startupjs/ui'
```

```jsx example
return (
  <Breadcrumbs
    routes={[
      {
        to: '#',
        name: 'StartupJs'
      },
      {
        to: '#',
        name: 'Docs'
      },
      {
        to: '#',
        name: 'Component'
      }
    ]}
  />
)
```


## Кастомный разделелитель

Разделитель по умолчанию - `/`. Его можно изменить, передав строку в свойство `separator`.

```jsx
import { Breadcrumbs } from '@startupjs/ui'
```

```jsx example
return (
  <Breadcrumbs
    separator='>'
    routes={[
      {
        to: '#',
        name: 'StartupJs'
      },
      {
        to: '#',
        name: 'Docs'
      },
      {
        to: '#',
        name: 'Component'
      }
    ]}
  />
)
```

## Хлебные крошки с иконками
Каждый элемент маршрута принимает свою собственную иконку в свойстве `icon`. Положение иконки можно изменить, передав в компонент `iconPosition` (`left` по умолчанию).

```jsx
import { Breadcrumbs } from '@startupjs/ui'
import { faHouseUser, faTable } from '@fortawesome/free-solid-svg-icons'
```

```jsx example
return (
  <Breadcrumbs
    routes={[
      {
        to: '#',
        name: 'House',
        icon: faHouseUser
      },
      {
        to: '#',
        name: 'Table',
        icon: faTable,
        iconPosition: 'right'
      }
    ]}
  />
)
```

## Размеры

Размер по умолчанию - `m`. Его можно изменить, изменив свойство `size`. Возможные значения свойства можно найти в разделе `Песочница` внизу страницы.

```jsx
import { Breadcrumbs } from '@startupjs/ui'
```

```jsx example
return (
  <View>
    <Breadcrumbs
      size='l'
      routes={[
        {
          to: '#',
          name: 'StartupJs'
        },
        {
          to: '#',
          name: 'Docs'
        },
        {
          to: '#',
          name: 'Component'
        }
      ]}
    />
    <Br />
    <Breadcrumbs
      size='s'
      routes={[
        {
          to: '#',
          name: 'StartupJs'
        },
        {
          to: '#',
          name: 'Docs'
        },
        {
          to: '#',
          name: 'Component'
        }
      ]}
    />
  </View>
)
```
